## 二、本周的几条新发现

### 1. 美国全生态系列：第三篇直接破 100k，vertical 仍在加速

教育 45k → 医疗 99k → 移民 103k。三连发，每一篇都比前一篇更大。

上周的 framework 修正"第一篇 ≥30k 时，第二篇按第一篇的 30-50% 估"在本周直接失效。移民没有按医疗 99k 的 30-50%（30-50k）跑，而是基本持平甚至略高。

原因有几条：
- **vertical 不会随两篇耗尽**。教育和医疗各自的核心受众池不同（家长 / 病人），移民又是另一波（H1B / 绿卡 / 入籍家庭）。三个池子有重叠但不重合，所以接力没有撞墙
- **强 claim 标题在大众痛点上的复利**。"一篇文章讲清楚 X" 这个模板第三次重复使用，读者已经认账号 = "这一类问题的总结者"，标题本身就是品牌
- **海外华人焦虑池的真实规模**远大于 framework 此前估计。教育 + 医疗 + 移民三大焦虑里，移民可能是最强的，因为不像教育（只关心娃）和医疗（不生病就不焦虑），移民身份焦虑是每天都在的

framework 应该把"美国主要系统全生态"vertical 的天花板从原本估计的 50-100k 上调到 100-200k，至少在前 5-7 篇之内不会衰减。

### 2. 嘘声四起 2.3k 这个反例：AI 焦虑题材，没生态绑定就跑不动

5/21 嘘声四起是本周唯一低于预测的，而且幅度大（-62%）。这一条要单独看。

按 framework 之前的归因，AI 焦虑大众池 + 具象 hook + 美国上下文 = 应该高。实际只 2.3k，比 5/5 "你会被 AI 取代吗" 4.1k 还低。原因可能有三条：

- **题材是"评论一个现象"而不是"梳理一个系统"**。教育全生态和医疗系统是"看完能用"的工具型科普，嘘声是"看完知道一件事"的评论型短文。X 上工具型 retention 远高于评论型
- **结论太开放**。"年轻人到底讨不讨厌 AI" 是个没有 closure 的命题。读者点开标题后发现没有 actionable 的判断，分享意愿低
- **跟英文版竞争**。同一天发了 Boos at Commencement 英文版，可能在 X 算法里把推荐 quota 分掉了一部分

修正：AI 焦虑题材在没有"教读者怎么办"的工具性收尾时，预测应该砍掉 50%。这条规律可以作为 framework 的新一条，跟"问号扣减"区分（问号扣减是标题层面，工具性收尾是内容层面）。

### 3. 物理 / 数学科普的天花板被上调

祖父积分学 5.7k + 量子计算机 9.7k + 数学史（上）7.4k（上周），三篇硬核 STEM 科普全部在 5-10k 区间。

framework 之前给 STEM 科普的预测一直在 2-5k，依据是"受众窄"。本周和上周数据说明，在账号势能高位时，STEM 科普的天花板可以推到 10k 左右。原因：
- 账号粉丝结构里硬核 STEM 受众的占比比 framework 预估的高
- "有前途吗" 这种带争议性的问句在窄受众里反而是好奇心驱动（"你来告诉我"），而不是悬疑扣减（"我懒得点"）

修正：STEM 科普 baseline 从 2-5k 上调到 5-10k。问号扣减在 STEM 受众池上也基本不起作用，跟大众痛点池一致。

### 4. 游戏 nostalgia + AI 实验，是一个未被识别的新 vertical

大航海时代 2 跑了 8.6k，是 framework 完全没有 comp 的题材。这一篇暴露出一个新 vertical 的存在：**80 后码农 / 工程师 nostalgia + AI 工具试验**。

共同特征：
- 受众是技术圈中 35+ 岁段，对老 IP 有情感
- 内容形式是"用 AI 工具去拆 / 复现一个老东西"
- 钩子来自 nostalgia 本身，AI 是手段不是主角

这个 vertical 的上限暂时不清楚，需要再 2-3 篇样本才能 calibrate。framework 暂时把它的天花板设为 5-15k。

## 三、整周偏差归因

5 篇里 4 篇上行偏差，1 篇下行偏差。按偏差幅度分组：

- **移民 103k +158%**：美国全生态 vertical 还在加速 + 标题品牌复利。framework 仍在低估这个池子的天花板
- **量子计算机 9.7k +223% / 大航海时代 8.6k +91% / 祖父积分学 5.7k +43%**：STEM / nostalgia 受众池被低估 + 账号势能高位
- **嘘声四起 2.3k -62%**：AI 评论型题材在无工具性收尾时的下行扣减没被算进去

整周方向上仍然是 framework 偏保守，但跟 5/11-5/15 的 +597% 比，本周偏差降到 +125%，主要因为没有"医疗 +1423%"这种极端 outlier。移民虽然 +158% 还是大，但绝对值在 100k 这个量级，已经是 framework 见过的状态。

- **继续低估全生态 vertical 天花板**。framework 应该把这一类的预测中位数 ×2-3，不再用历史 comp 做 anchor
