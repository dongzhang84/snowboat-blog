### 限流证据

数学照妖镜半小时 38 impressions 是关键信号。即使是上周最低的"三个 Claude Skill"（300+ X views，audience 极窄）首小时也不止 38。一个有 5k+ followers、过去一个月平均 3-10k impressions/篇的账号，半小时跑不出三位数 impressions，是平台层面的压制，不是内容问题。

前两篇也印证了同一信号。你会被AI取代吗题材是本周最大基数 + 最强情绪钩子，四支柱满分、framework 加分，正常 baseline 应在 6k 以上。1.2k 是基本盘的五分之一。大语言模型发展史 1.8k 也类似，AI 编程 4/27 同框架是 13k。

三篇连续偏差都在 70% 以上，且偏差量级单调放大（70% → 88% → 99%），不是单篇内容问题，是账号势能整体性衰减。

### 可能的限流触发原因

1. **题材敏感**：4/19 黄仁勋那篇 2k（pending-notes 当时记过首小时<100 views，靠社区分发救场）。这次的相关性 + AI + 中美话题可能持续触发限流权重
2. **频率**：4/27-5/6 十天发了 8 篇长文，账号被识别为高频内容农场也会触发降权
3. **链接**：每篇都带知乎和文章链接，外链密度高 X 算法不喜
4. **互动跌**：上一波 22k Vibe Reading 之后有个互动峰，峰后回落本身也会触发"势能衰减"

### 接下来怎么办

短期（本周内）：
- 周四的 创业方法论、周五的 Vibe Learning 暂停
- 等 24-48 小时观察账号是否自动恢复
- 测试性发一条短推（不带链接、不带长文），看 impressions 是否回到正常区间

中期（下周开始）：
- 频率降到 3 篇/周，给账号留呼吸空间
- 文章里的链接放评论区不放主帖

- 跟 5-10 个有交集的中型号互转，重建社交分发

需要验证：
- 限流是 7 天软限流还是更久
- 哪条触发因素占主因（题材 / 频率 / 外链）
