## 一·五、被 X 限流（这一份原本漏掉的关键事实）

写这份复盘的时候，原本完全没读pending-notes-2026-05-06。后来翻出来才发现5/6当天我记过一条关键观察：本周前三篇在首小时全部远低于预测，第三篇数学照妖镜半小时只有38 impressions。一个有5k+ followers、过去一个月单篇平均3-10k impressions的账号，半小时跑不出三位数，是账号层面被X限流。

这件事意味着上面"问号标题 -40%""同vertical两天即耗尽"这两条新规律的偏差量级**被高估**。本周三篇的预测偏差，至少有一半来自账号被限流，不全是标题问题或vertical耗尽问题。下周用这两条规律做预测时，应该把幅度减半。

可能的限流触发因素，按可能性排：

1. **频率太高**：4/27-5/6十天发8篇长文，账号被X识别为高频内容农场降权
2. **题材敏感**：本周混了"AI取代""数学AI"等关键词，可能触发权重压制
3. **外链密度**：每篇都带知乎链接 + 文章链接，X算法不喜外链密度高的账号
4. **峰后回落**：4/30 Vibe Reading 22k是个互动峰，峰后自然回落本身就会触发"势能衰减"

接下来一周的操作（已经在pending-notes里写过，这里固化）：

- 把发布频率从每天降到3篇/周，给账号留呼吸空间
- 文章链接放评论区不放主帖
- 长文跟短推交错发，避免连续同类内容
- 跟5-10个中型账号互转，重建社交分发

这是framework之前没有专门覆盖的维度：账号势能 / 限流状态是独立于内容质量的一条横向维度，应该作为预测时的乘数项加进来。

## 二、本周的几条新发现

### 1. 强比喻或类比的标题加分超出 framework 预测

数学照妖镜（5.5k）和福特经济学和AI经济学（5.4k）这两篇都不在大众痛点vertical，受众池本来都偏窄。但是它们都跑出了5k+，超出framework原来的预测。

共同特征是标题用了一个强比喻或强类比：照妖镜是给"AI检验数学"找了一个民间词汇的镜子。福特经济学是把"AI经济学"这个看上去抽象的问题，套到一个所有人都听过的历史类比上。这两种框架在X上的点击行为里，比预测的更强。

framework之前主要看身份锁定、数字反差、情绪钩子，没有专门给"标题里有强比喻 / 强类比"这种修辞加分。本周两次同方向证据，可以补一条规律。

### 2. 问号收尾的标题扣减比预想严重

你会被AI取代吗（4.1k）的预测是10k，实际只跑了4.1k，是本周最大的预测偏差。

framework在低流量失败模式里已经有"标题悬疑 / 双向态度"那一条（A模式），用骗子那篇0.8k做例子。但这篇标题"你会被AI取代吗?"四支柱全满（身份锁定超强、自救框架明确、数字反差、情绪焦虑），仍然被问号结尾压掉一档。

可能的机制是：X上的读者扫标题做1秒判断，问号让人立刻产生"看完不一定能解决我的焦虑"的预判，划走率明显高于宣言式标题。framework的A模式应该扩展到所有问号收尾的标题，不只是"是A还是B"双向悬疑。

### 3. 同 vertical 连续两天的耗尽效应再次出现

5/4大语言模型发展史 + 5/5你会被AI取代吗 都是AI大众焦虑vertical（前者是历史叙事，后者是诊断框架），两天连续触发同一拨"AI焦虑读者"。结果两篇都低于预测：前者 -38%，后者 -59%。

这跟4/27-4/29三连发AI圈内长文（13k → 3.1k → 3.9k）的衰减曲线一致。Framework已经有"题材轮换打破注意力耗尽"那条规律，但本周证据说明这条比预想更敏感：两天连续同vertical已经触发耗尽，不需要三天。

5/6数学照妖镜（AI + 数学）和5/7福特经济学（AI经济学）做了vertical切换，立刻把流量拉回5k+，跟这条规律吻合。

### 4. 系列品牌效应抗住了周末扣减一部分

Vibe Learning卡在BJ周六早（周末窗口），预测中位8-9k，实际11k，比中位高一档。这说明系列品牌效应（Vibe Writing 3.6k → Vibe Reading 22k → Vibe Learning 11k）能部分抵消周末扣减，但不能完全抵消（Vibe Reading是工作日早22k，Vibe Learning是周末早11k，单纯周末扣减大概拉了30-40%）。

Vibe系列接力的放大效应：第二篇相对第一篇 +500%（outlier量级），第三篇相对第二篇 -50%（回归均值）。后续如果继续做Vibe系列，单篇区间收敛到8-15k比较合理。

## 五、操作建议

### 该多发

- 强比喻 / 强类比的标题框架（数学照妖镜、福特经济学和AI经济学这一类）。即使题材本身窄，标题加成可以让窄题材跑出5k+
- 实用方法论 + 大众痛点（Vibe Learning系列、AI编程、AI学习路线图）。这条已经稳定证明天花板10k+，可继续延伸
- 切换vertical的发布节奏。两天同vertical已经耗尽，今后需要严格执行每天换一个vertical

### 该避免

- 问号结尾的标题。哪怕四支柱满分，问号会触发悬疑扣减，预测时要主动 -30% 到 -40%
- 同vertical连续两天发布。即使第二篇本身四支柱满分，前一天的注意力消耗会拉低30-50%
- 把"AI焦虑大众"vertical反复用。4/16、4/10已经各做过一次，5/5再做一次明显疲软。这条vertical需要2-3周间隔
- 高估audience pool size。AI历史叙事这一类只是AI内行兴趣，不是真正的"大众痛点"，pool中等

## 六、单条最高杠杆动作

如果只能改一件事：**预测时给问号收尾的标题主动 -40%，给两天同vertical的第二篇主动 -30%。**

这两条规律本周一起触发，把"你会被AI取代吗"从预测10k拉到实际4.1k，是本周方差最大的来源。整周预测加总只偏 -4%，单篇却能偏 -59%，主要就是因为这两条修正没有放进预测公式里。下周做预测，把这两条加进去，单篇方差应该能压缩一半以上。
