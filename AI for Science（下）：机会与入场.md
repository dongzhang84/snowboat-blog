# AI for Science（下篇）：机会与入场，科研 Agent 的水电煤，物理人怎么切

> 系列说明：这是 AI for Science 三篇系列的收官篇。上篇画了技术版图、给了诚实的成熟度判断。中篇跟着钱走，确认这是一个刚开门、资本在加码、但代价是慢的新赛道，并把它切成"够不着的造神层"和"够得着的水电煤层"。这一篇回答最后一个问题，对一个物理背景、却没有巨额资本的人，怎么进"科研 Agent 的水电煤"这一层。核心判断是，物理带来的是一套横切所有科学的通用武器，而不是某个小学科的深度，所以该建的，是任何科学 Agent 都得用的那层水电煤。

## 引言：在巨头造神的缝隙里

前两篇一直在"看"，看技术、看资本。这一篇要落到行动。

先把中篇的结论收紧成一句话。这扇门是开的，但它分层，最上面那层够不着，最下面那层够得着。造神层（自建实验室、造"AI 科学家"）要的是名人堂团队加数亿美元，对绝大多数人是橱窗里的样板。真正能进的是水电煤层，给科学 Agent 供底座、供评估、供工具、供数据的基础设施层。

为什么偏偏盯上这一层？因为它正热。这两年 Agent 本身已经卷到饱和，但给 Agent 做水电煤的活，给 Agent 当监控、当裁判、当客服的那些服务和平台，资本正大把涌进来。把这股热收窄到科学这个口子，就是"科学 Agent 的水电煤"。

但"水电煤够得着"还只是个方向，不是一刀。这一篇要把它磨成一刀，而磨刀的关键，是想清楚物理到底提供了什么独特的东西。

贯穿全篇的判断只有一条。物理提供的，是一套横切所有科学的通用能力，而非某个小学科的深度。所以在巨头造神的缝隙里，物理背景者最该占的位置，是"任何科学 Agent 都得用、且只有懂物理的人才做得好"的那层水电煤。这类活有一个共同特征，比拼的是学科判断力，不是模型规模。

下面六章这样走。第一章说清楚为什么水电煤是非巨头的主场。第二章是全篇的枢纽，讲物理那把横切武器到底是什么。第三章把横切的机会铺成一张菜单。第四章直面需求侧，学术界到底愿不愿意要。第五章看资本认不认这一层。第六章落到最具体的一步，第一刀切在哪、怎么从一篇论文走到能立住的形态。

## 第一章 机会缺口：为什么水电煤是非巨头的主场

### 1.1 先把造神层从选项里划掉

第一件事是诚实地承认，造神层进不去，别在它上面浪费幻想。

中篇说过，它的入场券是前 OpenAI、前 DeepMind 级别的履历加数亿美元起步资金。没有这两样，连第一轮都拿不到。更何况它的赌注最大、最不确定，要同时赌通"AI 真能自主做好科学""自动实验室真能稳定跑""烧出的数据真能喂出更强模型"三件事，一件不成就是数亿美元打水漂。对绝大多数人，造神层最大的用处是当风向标，看它成不成，比自己冲进去稳妥得多。

划掉造神层，谈不上认输，反倒是聚焦。把目光收回到够得着的那一层，机会反而更清楚。

### 1.2 四层基础设施，哪些够得着

上篇把基础设施拆成四层，科学基础模型（SciFM）、自驾实验室、评估/基准/可复现、工具与编排。从"小团队够不够得着"的角度重新排一遍，结论很明确。

- **SciFM（底座）：够不着。** 训练科学基础模型要重数据、重算力，是大机构和拿政府钱的联合体（如 NSF 和英伟达的开放科学大模型项目）的游戏。
- **自驾实验室：够不着。** 重资产、重设备，和造神层一样烧钱。
- **工具与编排：部分够得着。** 把某类科学数据库、某套仿真代码、某种设备接进 Agent，是可做的，但通用编排层竞争激烈，真正的机会在"领域特定的接口"。
- **评估/基准/可复现：最够得着。** 不要重资产、不要数亿美元，一个想清楚的小团队就能切进去，而且它恰恰需要别人没有的东西，学科判断力。

四层排下来，结论一目了然。对学科背景者，评估与工具这两层是性价比最高的入口，而且后面会看到，它们的边界正在模糊成一整片"给科学 Agent 验货、托底"的水电煤。这也和上篇"评估是最被低估、最关键的一层"的判断对上了。

### 1.3 一个筛选原则：挑"比模型规模、不如比学科判断"的位置

在往下钻之前，先立一把通用的筛子，任何一个具体机会都可以拿它过一遍。

**判断一个位置时，问一句，它的胜负主要由"模型有多强"决定，还是由"做的人有多懂科学"决定？**

如果是前者，绕开，那是大模型公司的主场，是拿短板拼别人长板。如果是后者，靠近，那是学科背景者能建墙的地方。出题、定标准答案、判对错、判一个结果在物理上可不可能，这些模型再强也替不了懂行的人。

这把筛子会贯穿后面几章。记住它，比记住任何一个具体案例都有用。

## 第二章 物理给的，是横切所有科学的通用武器

这一章是全篇的枢纽。前面说水电煤够得着，但很容易接着犯一个错，以为"那就回自己的学科，做天体物理的水电煤"。这一章要把这个错挑明，并给出一个大得多的视野。

### 2.1 一个反直觉的起点：别一头扎进自己的小学科

直觉会让人往最熟的地方钻。一个天体物理出身的人，第一反应就是做天体物理的评估、天体物理的工具。这没错，但它把人关进了一个很小的房间。

中篇那张学科版图说得清楚，天体物理在整个 AI4S 里钱最少、商业出口最远。如果水电煤只服务天体物理的 Agent，市场天花板就被这个小学科的天花板锁死了。

真正的破法，是换一个角度看自己的优势。物理背景者最值钱的东西，与其说是"懂天体物理"这一个学科的深度，不如说是物理训练给的那套通用能力，而这套能力横切所有定量科学。想通这一点，就不必待在一个小学科的房间里，而能站在所有科学 Agent 的门口。

### 2.2 物理的四样横切资产

物理训练真正值钱、且纯 CS/ML 背景的人难补的，是四样可迁移的能力。它们的共同点是，不绑定任何具体学科。

一是**一致性判断**。守恒律、对称性、量纲、单位、极限行为，这些是所有定量科学共享的硬约束。一个结果违不违反能量守恒、量纲对不对、在极限情形下退不退化到已知答案，物理人一眼能看出来，这是物理训练最底层、也最通用的直觉。

二是**不确定性量化**。一个测量、一个预测，到底有多可信，误差怎么传播，统计力学和机器学习本就数学同源，物理人对"如何在高维、含噪、不确定的空间里做推断"有天然的手感。

三是**仿真与建模**。把一个混乱的真实系统抽象成一个能算的模型、能跑的仿真，这是物理的看家本领，而且不挑学科，材料、气候、流体、天体都用同一套建模思维。

四是**最优实验设计**。给定有限的预算，下一个最该测什么、最该算什么，这是贝叶斯实验设计和主动学习的地盘，也是物理和统计的交叉强项。

这四样，恰恰是纯 CS/ML 背景的人最缺的。而下面会看到，每一样都能横切到一大片科学 Agent 上去。

### 2.3 把枪口对准"所有科学 Agent"

把这四样资产和"科研 Agent 的水电煤"叠在一起，新的落点就出来了。

材料 Agent 跑出一个结构，要有人判它物理上合不合理（一致性）。气候 Agent 给一个预测，要有人量它多可信（不确定性）。任何自驾实验室的 Agent，都要有人帮它决定下一步测什么（实验设计）。所有这些需求，都不绑定某一个学科，它们是科学 Agent 的通用刚需，而满足它们靠的正是物理那四样横切资产。

所以这一篇接下来要说的，是一件大得多的事。用物理那套横切能力，去建任何科学 Agent 都要用的水电煤。这样既守在水电煤这条又热又有钱的火线上，又不被锁进任何一个小学科，物理也从一个学科标签，变回了一把通用武器。

下一章，就把这片横切的机会，铺成一张具体的菜单。

## 第三章 机会地图：横切的科研 Agent 水电煤菜单

这一章把机会摊开。组织的方式是两个轴，纵轴是"这块水电煤站在 Agent 流水线的哪个位置"，横轴是"它主要吃哪一样物理资产"。下面按位置分五类，每一类都尽量横切多个学科。

### 3.1 守门检验类：给所有科学 Agent 当一致性裁判

这一类的共同思路，是在 Agent 出结果的那一刻拦一道，问它"这在物理上站得住吗"。

- **物理一致性检验器。** 任何科学 Agent 跑出个结果，扔给它问一句，违不违反守恒、对称、已知物理边界。这是把"守门人"做成横切的，不守一个学科的门，守所有定量科学的门。
- **单位与量纲的校验中间件。** 专抓 Agent 最经典的错，单位混用、量纲不对。小到可以就是一个工具，但每个定量科学 Agent 都得挂，几乎零竞争。
- **数量级与极限的"离谱"检测。** Agent 给个数，先问它和已知尺度差了几个数量级，或者在某个极限下退不退化到已知答案。物理人天生是费米估计高手，这是纯 CS 人没受过的训练。
- **物理驱动的"假科学"检测器。** 用物理一致性去识别 AI 编造的、看着合理却违反物理的结果。这一个直接踩在第四章要讲的痛点上，科学界正被 AI 生成内容淹没，急需一面识别真假的盾牌。

### 3.2 不确定性类：给所有科学 Agent 标一个可信度

这一类吃的是第二样资产，不确定性量化。

- **不确定性量化（UQ）服务。** Agent 给个答案，它到底有多可信，做成一个任何科学 Agent 都能挂上去的层。
- **误差传播引擎。** Agent 把好几步算法串起来时，把每步的不确定度顺着链条传下去。上篇那个"复合失败"有个定量的兄弟，就是误差越串越大，而误差传播是计量物理的老本行。
- **校准检测。** Agent 嘴上说"九成把握"，实际真有九成吗，查它的置信度准不准，横切所有学科。

### 3.3 训练场与奖励类：资本最热的一块

这一类吃的是第三样资产，仿真与建模，而且正好踩在"训练 Agent"这个当下最烧钱的基础设施上。

- **物理仿真的训练与评估环境。** 现在的科学 Agent 要在环境里训练和被考核，而物理仿真是造这些训练场和考场的通用底料。SciAgentGym、SimulCost、Stargazer 这些工作，就是这条线的早期苗头。
- **"可验证奖励"的科学 verifier 库。** 现在训 Agent 最主流的方法，是给它可验证的奖励，而科学这边的 verifier 严重缺货，谁来判一个推导、一个结果是不是真满足物理。这层是所有训练科学 Agent 的人都要用的水电煤，又纯粹是物理判断。
- **可控难度的合成任务工厂。** 用物理仿真当标准答案机，批量生成有真值、难度可调、还能塞"分布外"陷阱的科学任务，给所有人训练和考核科学 Agent。Gravity-Bench 那种"往模拟里塞个改过的物理定律"的思路，可以做成一条通用产线。

### 3.4 决策类：自驾实验室的大脑

这一类吃的是第四样资产，最优实验设计。

- **最优实验设计与主动学习中间件。** 给定预算，下一个该测什么、该算什么，这是所有自驾实验室和数据采集 Agent 的大脑。它横切材料、生物、天文，因为每一类都要决定下一步做什么，而它又是贝叶斯实验设计、物理和统计的甜区。上篇就点过，"高维实验设计加不确定性管理"是物理背景者的天然甜区。这块资本密集（自驾实验室正烧钱），却几乎没人把它单独做成一层。

### 3.5 接口与复现类：把仿真和复现接给 Agent

这一类偏工程，但同样吃物理。

- **仿真器的标准化 Agent 接口。** 把 DFT、分子动力学、流体、N-body 这些求解器，包成干净、带物理报错、Agent 能稳定调用的工具。把仿真接给 Agent 用这件事，做对了就是横切所有仿真密集科学的水电煤。
- **可复现验证与注入错误检测。** 通用的"这个 AI 科学结果换人换次能不能重现"，以及"能不能查出被人塞进去的错"。

### 3.6 哪几刀最锋利

菜单很长，但如果只押少数几个，最值得押的是这三个，因为它们横切最广、最贴当下钱口、又最吃物理判断这把刀。

一是 **3.3 的科学 verifier 库**，踩在"训练 Agent"这个最热的基础设施上，科学 verifier 又奇缺，纯学科判断。二是 **3.4 的最优实验设计中间件**，自驾实验室的大脑、资本密集，横切所有要决定下一步的科学 Agent。三是 **3.1 的假科学检测器**，最贴第四章那个"科学界要盾牌"的当下痛点，叙事最性感，也最像守门人该干的事。

还要单独说一句"评估/基准"。它在这张菜单里只是一类，散落在 3.1 到 3.3，不再是唯一主菜。原因是，把眼界从"做一个学科的基准"放大到"建横切的水电煤"之后，评估只是横切能力的一种用法，而 verifier、实验设计、UQ 这些，市场更大、也更难被一个小团队抢先占满。

## 第四章 需求侧：学术界真的愿意要吗

前三章都在讲"能供什么"。但有一个绕不开、也最容易被乐观情绪盖过的问题，需求侧到底愿不愿意要。这一章必须直面它，因为很多 AI 进入专业领域的项目，死掉的原因不在技术，在于那个领域的人根本不买账。

### 4.1 一个真实的教训：替代型应用为什么被反感

先讲一个许多人踩过的坑。把 AI 做成"替代专业人士"的工具，结果遭到强烈抵触，项目做不起来。

这类失败的根源往往不在技术，在于定位站错了边。当一个产品被那个领域的人理解为"要取代我、威胁我的饭碗、冒犯我这门手艺"，他们的本能反应就是抵触，无论技术多好。教育领域的不少 AI 尝试就栽在这里，老师把它当成替代者，而不是帮手。

这个教训对 AI4S 极其重要，因为它直接决定了该站哪一边。

### 4.2 科学界对 AI：又用又怕

把镜头转向科学界，会看到一幅复杂的图景，研究者一边在用 AI，一边在怕 AI，而且"怕"得很具体。

一方面，AI 工具被广泛使用。另一方面，抵触和担忧同样真实。有调查显示，相当多研究者怀疑 AI 生成内容的准确性与完整性。还有一层更深的焦虑，当 AI 能在关键研究任务上逼近甚至超过人类，"如果 AI 能替我做，我还剩什么"成了一个真实的、影响采纳意愿的问题。学术生态层面，AI 生成内容的泛滥已经造成实质冲击，AI 生成的综述大量涌现，以至于 arXiv 在 2025 年底收紧了计算机方向综述类文章的投稿，某顶会被报道有约两成的同行评审由 AI 生成，形成"AI 写、AI 审"的隐忧，隐藏提示词操纵评审等新型不端也开始出现。

一句话，科学界对"AI 替我做研究"是警惕甚至抵触的，这一点和教育界并无本质不同。一个"替代科学家的 Agent"，会撞上同样硬、甚至更硬的墙。

### 4.3 关键 reframe：守门人，而不是替代者

但科学界"怕 AI"这件事，恰恰为水电煤这一层创造了需求，这是 AI4S 和教育那类失败最本质的区别。

逻辑是这样的。科学界正被 AI 生成内容淹没、被分不清真假困扰、被"AI 写 AI 审"的闭环威胁、被加剧的可复现危机折磨。他们最急需的是什么，是能帮他们辨别、验证、把关 AI 的工具。而第三章那张菜单上的东西，假科学检测、一致性检验、UQ、可复现验证，正好就是这样的工具。

这就把定位彻底翻转了。做这层水电煤，是站在科学家这边、给他们递一面对抗 AI 泛滥的盾牌，而不是抢他们的饭碗。需求方甚至在主动呼唤，大型科学合作组已经公开把"验证框架"列为优先级、直言"评估框架才刚开始出现"、呼吁建立领域专属的评估基准。这里面对的，是一个焦虑的、主动在找人补位的市场，而不是一个需要苦苦说服的抵触市场。

而且横切的好处在这里又显出来了。这面盾牌不绑定某一个学科，任何学科都能用，需求面比单学科评估宽得多。

### 4.4 一条死守的定位红线

把这章收成一条必须死守的红线。**做盾牌，不做替代。**

具体说，做"帮科学家把关、验证、复现 AI"的工具，而不是"替科学家做研究、抢科学家署名"的系统。前者科学界欢迎、需求方在主动找，后者会重摔一次"被专业群体集体抵触"的跟头。这条线，从产品的第一天起就要守住，因为它一旦站错，再好的技术也救不回来。

## 第五章 资本会不会认这一层

需求侧搞清楚了，还要回答资本侧，水电煤这层，钱认不认。答案要分两层说，避免被乐观或悲观任何一边带偏。

### 5.1 好消息：评估作为大类，已被验证能融大钱

先看大类。"做评估能不能成为一门大生意"这个问题，资本已经用真金白银回答了，能。最硬的例子是一个从大学众包打分项目独立出来、专做大模型评估与排行榜的公司（LMArena），累计融资数亿美元、估值奔着二十亿去。一个"给模型打分"的平台能值这么多，说明评估本身就是被资本认可的、能融大钱的方向，而不是无足轻重的配套活。

### 5.2 冷静面：科学评估这个细分，对纯 VC 还偏早

但要把话说全。那个高估值的评估公司，评的是通用大模型，受众极广、商业化路径清楚。一旦细化到"科学评估"，给某个学科的科研 Agent 出考卷，受众一下子窄了很多，而且现在托底这块的，主要还是政府和公益的钱（能源部、国家科学基金会、公益基金会）。换句话说，科学评估作为一项严肃的学术与基础设施工作是真实的，但作为一门纯商业的风投生意，眼下还偏早。

### 5.3 含义：先学术，后商业

这个"大类已验证、细分尚早"的组合，恰恰指向一条最适合学科背景者的路径，学术先行，而不是一上来就纯 VC 创业。

理由很顺。科学评估现在更像一块"先靠公益和学术养着、等商业化拐点"的地。对学科背景者，这反而是好事，他们本来就在学术体系里，发一篇扎实的基准论文、把它开源、建立起领域内的声誉，是天然擅长、且风险极低的动作。等这块地的商业化拐点到来，参照通用评估已经走通的路，此时已经握着声誉和标准制定权，再谈服务化、商业化或被并购，都顺理成章。这条路径的具体走法，是第六章的最后一块。

## 第六章 入场路径：从一篇论文到立住一个横切标准

前五章给了方向、菜单、定位和资本判断。这一章落到最具体的一步。一个物理背景的人，第一刀到底切在哪、怎么走。

### 6.1 选第一刀：挑一个"只有懂物理的人能做、且横切"的切口

回到第一章那把筛子，再叠加这一篇的横切主线，选第一刀的标准就两条。一是只有懂物理的人才能做好（比学科判断不比模型规模），二是它能横切到多个学科，不被锁进一个小房间。

按这两条，第三章那三个最锋利的切口都合格，假科学检测、科学 verifier 库、最优实验设计。挑哪个，看最想先攻哪块，但有一个务实的起步技巧。

先在自己最熟的领域，把它跑通一次。以天体物理背景为例，那就拿天体物理当第一块试验田，因为在这里出题最准、判分最稳、阻力最小。但天体物理只是练手场，终点在更宽的地方。第一个东西从设计的第一天起，骨架就要是横切的，方法要能原样搬到材料、气候、生物的 Agent 上去。

### 6.2 三步走

把练手和横切合成一条路径。第五章说过，科学水电煤"大类已验证、细分尚早"，所以最稳的走法是学术先行。具体三步。

**第一步，在最熟的领域发出第一个，开源、立声誉。** 比如拿天体物理做一个横切设计的 verifier 或一致性检验器，配一篇基准论文和公开 leaderboard。这一步几乎不需要资本，就能把"做科学 Agent 水电煤"这件事跑通，并在领域内立住名声。它的产物不只是论文，更是一套可复用的方法论，一套"怎么给科研 Agent 出题判分、验货把关"的打法。

**第二步，把同一套能力横展到更多学科、更热的方向。** 用第一步攒下的方法论和声誉，把这套横切能力铺到材料、气候这些钱景更好的方向去。这一步进入的是更大的市场，靠的是横切武器的可迁移性，不必为每个学科从头补课。

**第三步，成为"科学 Agent 可靠性"的事实标准，等商业化拐点。** 让做出来的检验器、verifier 或实验设计层被引用、被采用、最好被造 Agent 的公司和实验室纳入流程。一旦它成了大家默认用来衡量"科学 Agent 靠不靠谱"的那把尺子，就握住了标准制定权。参照通用评估已经走通的路，从开源项目到数亿美元估值的公司，等这块的商业化拐点到来，可以从容选择，做成持续服务、做认证、做私有定制，或被需要这套能力的公司并购。

### 6.3 为什么这条比"主攻一个小学科"强

最后说清楚，为什么这条横切路线，比"主攻材料评估"或"主攻天体评估"都强。

一是不被锁进小学科。市场天花板不再被某一个学科封死，而是所有科学 Agent。二是物理当通用武器。那四样横切资产可以反复复用，做一次、卖很多学科，不必为每个学科从头补课。三是它最接近学科背景者该走的路子，靠一篇篇扎实的研究把声誉立住，让机会反过来找上门，而不是一上来就赌一个重资产的大局。

它慢，但稳，而且越做越宽。这正是给一个有学科背景、没有巨额资本的人量身的走法。

## 结语：守门人，还是造神者

三篇走到这里，可以把整个系列收成一个选择。

上篇说，AI 正在从"读科学"走向"做科学"，范式转变是真的，但它做得还很不可靠。中篇说，资本已经认了这个方向、门是开的、钱在加码，但代价是慢，而且这扇门分层。下篇说，对一个有学科背景、却没有巨额资本的人，能进、且该进的，是水电煤层里那些"只有懂科学的人才能做、巨头又懒得做"的活，而且最好把物理当成一把横切所有科学的通用武器，去建那些任何科学 Agent 都要用的检验、验证、训练设施。

所以最后的选择，其实是一个身份问题，造神者，还是守门人。

造神者要造出"AI 科学家"，那是名人堂团队加数亿美元的豪赌，也是最可能先爆的雷。守门人不造神，他给这场造神运动当裁判、当验货员、当那面让科学界对抗 AI 泛滥的盾牌。守门人不需要数亿美元，靠的是别人翻不过去的学科判断力。他站在科学家这边，不跟他们抢饭碗。他守的是所有定量科学的门，而不只是一个学科的门。

回到上篇开头那个画面，AI 自己设计出能用的分子、自己合成出新材料。那是造神者的高光时刻。但每一个这样的高光背后，都需要有人问一句冷静的话，它真的做对了吗，还是只是看起来做对了。能可靠地回答这个问题的人，在未来很长一段时间里，都会是稀缺的。

对一个懂科学的人来说，这或许不是最耀眼的位置，但很可能是最适合、最扛得住、也最难被取代的位置。门开着，钱在加码，而最锋利的那一刀，恰好需要的就是懂科学的人手里那把别人没有的尺子。
