# 一、引言

中篇结束的地方，是1985年。

那一年，坎德拉斯（Philip Candelas）、霍洛维茨（Gary Horowitz）、斯特鲁明格（Andrew Strominger）和威腾（Edward Witten）四个人联合发表了一篇论文，标题是《超弦的真空结构》（*Vacuum Configurations for Superstrings*），发表在 *Nuclear Physics B* 上。这篇论文论证了一件事情：超弦理论所要求的"多出来的六维"，必须藏在卡拉比—丘成桐定理所描述的空间之中。

卡拉比—丘空间从此正式登上了物理学的舞台。

但是，登上舞台之后呢？卡拉比—丘空间在弦论里究竟扮演什么角色？它不只是"满足要求的一个空间"，不只是一个被动的"容器"。弦论要求的10维时空，4维是我们熟悉的相对论时空，另外6维蜷缩在卡拉比—丘空间里。这意味着，我们观测到的一切低能物理，包括粒子的种类、质量、相互作用，都由这个蜷缩的卡拉比—丘空间的几何形状决定。

换句话说，卡拉比—丘空间是弦论和现实物理世界之间的翻译器。

本篇的任务，就是把这个翻译的过程说清楚。然后我们会看到，这个翻译器带来了一个意想不到的困境：卡拉比—丘空间的数量相当庞大，远非唯一。弦论的"终极理论"梦想，也因此笼上了一层阴影。

# 二、从几何到物理：卡拉比—丘空间决定了什么

## 2.1 霍奇数：空间有多少种"变形方式"

卡拉比—丘空间的几何，可以用一组数来描述。这组数叫做霍奇数（Hodge numbers），记为 $h^{p,q}$，是以英国数学家威廉·霍奇（William Hodge, 1903—1975）命名的。

霍奇是苏格兰人，在剑桥大学长期任教。他在1941年提出了著名的"霍奇猜想"，至今未被证明，是克雷数学研究所悬赏的七个千禧年大奖问题之一（每个问题奖金100万美元）。霍奇把代数几何和微分形式联系了起来，给复流形上的微分形式做了一套精密的分类。这套分类，就是霍奇数的来源。

要理解霍奇数的直观含义，可以这样想：一个卡拉比—丘空间虽然看不见摸不着（至少六维，人眼无法直观想象），但是它有一定的"形状"。这个形状可以发生变化，就像一个橡皮泥捏的物体可以被拉伸、压缩、扭曲一样。霍奇数告诉我们的，就是这个卡拉比—丘空间有多少种独立的变形方式。

对于复三维的卡拉比—丘流形（也就是弦论最需要的那种，实六维），最重要的两个霍奇数是 $h^{1,1}$ 和 $h^{2,1}$。

$h^{1,1}$ 刻画的是卡拉比—丘空间的"大小和膨胀模式"（Kähler模）。打个比方，如果卡拉比—丘空间内部有若干个"洞"或者"环路"，$h^{1,1}$ 告诉你这些洞有多少种独立的方式可以被"吹大"或者"压小"。在物理上，$h^{1,1}$ 对应着低能理论里某一类无质量标量场（与轴子相关）的数目。

$h^{2,1}$ 刻画的是卡拉比—丘空间的"形变模式"，也就是空间形状可以在多少个独立方向上发生"扭动"。在物理上，$h^{2,1}$ 对应着另一类无质量标量场（模量场）的数目。

通俗一点说：$h^{1,1}$ 管"大小"，$h^{2,1}$ 管"形状"。两者加起来，就完整地描述了卡拉比—丘空间在几何上的自由度。

## 2.2 欧拉示性数与费米子的世代数

上篇花了很大的篇幅介绍了欧拉示性数 $\chi$。我们知道，欧拉示性数是拓扑学的基石，从多面体的 $V - E + F = 2$ 开始，一路发展到陈省身—高斯—博内公式，成为了刻画空间整体拓扑性质的核心不变量。

对于卡拉比—丘三维流形，欧拉示性数和霍奇数之间有一个简洁的关系：

$$\chi = 2(h^{1,1} - h^{2,1})$$

这个公式把上篇的数学（欧拉示性数、陈类）和本篇的物理（霍奇数、卡拉比—丘空间的几何自由度）直接联系了起来。

但是，这个公式的物理意义远比它的数学形式更加震撼。

1985年的坎德拉斯等四人论文证明了：在杂化弦论（heterotic string theory）的框架下，将自旋联络嵌入规范联络，$E_8$ 群被破缺为 $E_6$ 群，此时卡拉比—丘空间的欧拉示性数的绝对值的一半，$|\chi|/2$，恰好对应弦论低能极限里费米子的净代数（net number of generations），也就是"代数"减去"反代数"。

我们知道，标准模型里的费米子有三代：

- 第一代：上夸克、下夸克、电子、电子中微子
- 第二代：粲夸克、奇夸克、渺子、渺子中微子
- 第三代：顶夸克、底夸克、陶子、陶子中微子

为什么恰好是三代？标准模型对此没有任何解释。三代费米子是实验给出的既成事实，就像上帝随手写下的一个数字，物理学家只能接受。

而弦论给出了一个几何上的解释：如果 $|\chi|/2 = 3$，即卡拉比—丘空间的欧拉示性数为 $\pm 6$，则弦论预测恰好有三代费米子。

让我再强调一遍这句话的分量。上篇里，我们从欧拉的多面体公式 $V-E+F=2$ 出发，经过高斯曲率、高斯—博内公式、陈省身的推广，一路走到了欧拉示性数这个拓扑不变量。现在这个不变量告诉我们，它决定了我们宇宙里基本粒子的世代数。

这是整篇文章最关键的一句话，也是上篇数学和下篇物理之间最直接的一次握手：

> 陈省身—高斯—博内公式里的欧拉示性数，决定了我们宇宙里费米子的世代数。

丘成桐第一次听到这个对应关系的时候，感到了数学家特有的那种兴奋。他在《大宇之形》中回忆过这种感受：自己花了多年证明的一个纯数学定理，居然能和现实世界的基本粒子发生如此直接的联系。

数学不只是"有用"的问题，而是"这也太巧了"的问题。这种巧合，要么是偶然的（那就太诡异了），要么是暗示着一种更深层的结构（那就太美了）。

物理学家们很快追问了一个尖锐的问题：欧拉示性数为 $\pm 6$ 的卡拉比—丘空间有多少种？

答案是：很多种。

这个"很多种"在后面会变成一个大问题。但在1985年的兴奋中，大家暂时顾不上这个。

## 2.3 几张表格，物理和数学的对应关系

在讲卡拉比—丘空间和物理的对应之前，先做一个更根本的铺垫：物理学家用的规范场论，和数学家用的纤维丛理论，本质上是同一个东西。

1954 年，杨振宁和罗伯特·米尔斯（Robert Mills）在研究强相互作用时，提出了后来被叫做"杨-米尔斯规范场论"（Yang-Mills gauge theory）的数学框架。他们以为自己发明了一套全新的几何工具。直到 1970 年代初，杨振宁才意外发现：这套工具，数学家陈省身和他的前辈们在 1940-1950 年代已经完整发展出来了，叫做纤维丛（fiber bundle）理论。

1975 年，杨振宁和吴大峻（Tai Tsun Wu）在 *Physical Review D* 发表论文 *Concept of Nonintegrable Phase Factors and Global Formulation of Gauge Fields*，第一次把这两套语言并列在一张翻译字典上。这张字典在物理圈被叫做 Wu-Yang Dictionary（吴-杨字典）。

对应关系

| 物理（杨-米尔斯 规范场） | 数学（陈省身 纤维丛） |
|---|---|
| 规范势 $A_\mu$ | 主丛上的联络 |
| 场强 $F_{\mu\nu}$ | 联络的曲率 |
| 规范群 $G$ | 主丛的结构群 |
| 规范变换 | 丛的局部同构 |

一句话：物理学家 1950 年代用来描述电磁场、后来用来描述弱力强力的那套数学结构，数学家在 1940 年代就已经沿着自己的路径发展出来了。双方用的是同一套方程，研究的是同一批几何对象，只是名字完全不同。

详细版对应（Wu-Yang 字典）：

| 物理概念 | 数学概念 |
|---|---|
| 规范势 $A_\mu$（gauge potential） | 主丛 $P$ 上的联络 1-形式 $\omega$ |
| 场强 $F_{\mu\nu}$（field strength） | 曲率 2-形式 $\Omega = d\omega + \omega \wedge \omega$ |
| 规范变换（gauge transformation） | 主丛的自同构 |
| 规范群 $G$（gauge group） | 主丛的结构群 |
| 物质场 / 源 | 相伴丛（associated bundle）上的截面 |
| 电荷 / 拓扑量子数 | 陈类（Chern class） |
| 磁单极（magnetic monopole） | 非平凡主丛（non-trivial principal bundle） |
| Wilson loop | 沿闭路的和乐（holonomy） |
| Aharonov-Bohm 相位 | 非平凡联络的和乐 |
| 杨-米尔斯规范场论 | 非阿贝尔主丛上的联络几何 |

杨振宁自己后来讲过一段被广为引用的对话。他第一次搞清楚这张字典之后，跑去问陈省身："你们数学家怎么能凭空想象出这些概念？"陈省身的回答是："不，这些概念不是凭空想象的。它们是自然的、必然的。"

两条互不知情的路，一条从物理出发，一条从数学出发，最后走到了同一个地方。

理解了 Wu-Yang 字典之后，再看卡拉比—丘空间和低能物理的对应就顺了很多。卡拉比—丘空间自带一个 $SU(3)$ 和乐群，在 Wu-Yang 字典里这对应于规范场论里的 $SU(3)$ 对称性。弦论用卡拉比—丘空间做紧致化，得到的 4 维低能理论自然包含 $SU(3)$ 相关的规范结构。

卡拉比—丘空间的几何数据和低能物理的对应关系，可以粗略地总结如下：

| 卡拉比—丘空间的几何 | 低能物理 |
|---|---|
| $h^{1,1}$（Kähler模的数目） | 某类无质量标量场的数目 |
| $h^{2,1}$（形变模的数目） | 另一类无质量标量场的数目 |
| $\|\chi\|/2 = \|h^{1,1} - h^{2,1}\|$ | 费米子净世代数 |
| $SU(3)$ 和乐群（holonomy） | 保留 $\mathcal{N}=1$ 超对称 |

卡拉比—丘空间的形状，决定了我们看到的粒子世界。

这张表格看起来简洁，但它的背后是几十年的数学和物理积累。从高斯的曲面论到陈省身的示性类，从卡拉比猜想到杨振宁的规范场论，所有这些线索，在卡拉比—丘空间这个概念上汇合了。

## 2.4 理论很美，但实验呢？

表格里的对应关系确实漂亮。但是物理学不是数学，光漂亮不行，得和现实世界对得上。

最直接的检验是：有没有人找到一个具体的卡拉比—丘空间，既给出恰好三代费米子（$|\chi|/2 = 3$），又给出正确的规范群（标准模型的 $SU(3) \times SU(2) \times U(1)$），又给出正确的粒子质量和耦合常数？

答案是：至今没有完全成功的。

欧拉示性数为 $\pm 6$ 的卡拉比—丘空间确实存在，数量还不少。但是"三代费米子"只是标准模型的一个特征，标准模型还有19个以上的自由参数（各种粒子的质量、耦合常数等等）。要从一个卡拉比—丘空间出发，同时复现所有这些参数，目前做不到。

有些候选者"接近"了。比如，一些基于 $E_8 \times E_8$ 杂化弦论的紧致化方案，可以给出三代费米子和接近标准模型的规范群。但是"接近"和"精确"之间的距离，在物理学里往往就是一条鸿沟。

更尴尬的是，弦论预言了超对称粒子的存在。中篇里提到过，超对称预言了大量超越标准模型的新粒子。到目前为止，欧洲的大型强子对撞机（LHC）在 $10^4$ GeV 的能级上，没有发现任何超对称粒子。超夸克（squark）的质量下限已经被推到了1.1 TeV以上，胶微子（gluino）也被排除了1 TeV以下的可能性。

弦论的支持者说：超对称粒子可能藏在更高的能标上，LHC的能量不够。批评者说：一个理论如果永远可以把预言推到"更高的能标"，那它还是科学吗？

这个争论我们留到第五章再展开。眼下先记住一件事：卡拉比—丘空间和费米子代数之间的对应关系，在理论上是优美的，但在实验上还远远没有被验证。

# 三、镜像对称：弦论反哺数学

## 3.1 *一个有趣的发现*

1985年的四人论文之后，大量物理学家开始认真研究弦论在卡拉比—丘空间上的紧致化。很快，他们发现了一个令人惊讶的现象。

最早注意到这个现象的是物理学家兰斯·迪克逊（Lance Dixon）。1988年，迪克逊观察到一种可能性：两个拓扑结构完全不同的卡拉比—丘空间，有时候会给出完全相同的弦论物理。

1989年，沃尔夫冈·勒尔歇（Wolfgang Lerche）、库姆伦·瓦法（Cumrun Vafa）和尼古拉斯·沃纳（Nicholas Warner）进一步发展了这个观察。他们发现，IIA型和IIB型弦论分别在两个不同的卡拉比—丘空间上紧致化，可以产生等价的物理。这两个卡拉比—丘空间互为"镜像流形"（mirror manifolds）。

具体地说，如果一个卡拉比—丘空间的霍奇数是 $(h^{1,1}, h^{2,1})$，那么存在另一个卡拉比—丘空间，其霍奇数恰好是 $(h^{2,1}, h^{1,1})$，两个霍奇数互换，而这两个空间给出完全相同的弦论物理。

1990年，布莱恩·格林（Brian Greene）和罗南·普莱瑟（Ronen Plesser）找到了第一对非平凡的显式镜像对，通过研究卡拉比—丘流形和所谓Gepner模型之间的关系构造出来。镜像对称从一个观察变成了一个有实例支撑的理论。

这两个互为镜像的卡拉比—丘空间，被称为一对镜像（mirror pair）。

为什么这件事令人惊讶？因为根据欧拉示性数的公式 $\chi = 2(h^{1,1} - h^{2,1})$，两个镜像卡拉比—丘空间的欧拉示性数符号相反。一个是 $+\chi$，一个是 $-\chi$。在拓扑学里，欧拉示性数不同的空间是完全不同的空间，就像球面和救生圈是完全不同的东西一样。

然而在弦论的眼里，这两个拓扑上截然不同的空间，物理上是等价的。

这在纯数学里是完全没有预兆的。没有任何数学定理提示过，两个欧拉示性数符号相反的卡拉比—丘流形之间应该存在某种深层联系。是物理学，具体来说是弦论，第一次揭示了这种联系。

## 3.2 坎德拉斯的计算

镜像对称最惊人的应用，发生在1991年。

数学上有一个古老的计数问题，属于代数几何里的所谓"计数几何"（enumerative geometry）。问题是这样的：在一个五次曲面（quintic threefold）上，有多少条给定次数的有理曲线？

所谓"五次曲面"，是四维复射影空间 $\mathbb{CP}^4$ 里由一个五次齐次多项式定义的三维（复）超曲面。这个空间恰好是一个卡拉比—丘流形，是卡拉比—丘空间家族里最简单也最经典的一个例子。

"有理曲线"可以粗略地理解为"可以用有理函数参数化的曲线"，在代数几何里有精确的定义。

对于次数为1（也就是直线）的情况，答案是2875条。这个数是19世纪的德国数学家赫尔曼·舒伯特（Hermann Schubert）在1879年算出来的。舒伯特是计数几何的先驱，他发展的"舒伯特演算"（Schubert calculus）是代数几何里的经典工具。

对于次数为2（也就是二次曲线、圆锥曲线）的情况，数学家费了很大力气。1986年，美国数学家谢尔顿·卡茨（Sheldon Katz）算出了答案：609,250条。从舒伯特到卡茨，这一步跨了超过一个世纪。

对于次数为3的情况，数学家们正在努力计算中，挪威数学家艾灵斯鲁德（Geir Ellingsrud）和斯特鲁默（Stein Arild Strømme）也在攻关这个问题。

1991年，物理学家坎德拉斯和他的合作者德拉奥萨（Xenia de la Ossa）、格林（Paul Green）、帕克斯（Linda Parkes）发表了一篇论文，标题是《一对卡拉比—丘流形作为精确可解的超共形理论》（*A Pair of Calabi-Yau Manifolds as an Exactly Soluble Superconformal Theory*）。在这篇论文里，坎德拉斯等人用镜像对称，把五次曲面上的曲线计数问题转化到了镜像卡拉比—丘空间上的一个等价计算（周期积分），然后用一个相对简单的公式，一口气给出了所有次数的答案。

其中，次数为3的答案是 317,206,375 条。

这个数字公布后，立刻引起了一场风波。

1991年5月，在伯克利的MSRI（数学科学研究所）举办了一场会议，物理学家和数学家坐在一起讨论镜像对称。坎德拉斯报告了他的结果。几乎同时，挪威数学家艾灵斯鲁德和斯特鲁默也用传统的代数几何方法算出了次数为3的答案，但他们的数字和坎德拉斯的不一样。

在场的数学家们倾向于相信自己人。毕竟，坎德拉斯用的"镜像对称"并没有严格的数学证明，只是一种物理直觉。而艾灵斯鲁德和斯特鲁默用的是正经的代数几何工具。

然而几天后，艾灵斯鲁德和斯特鲁默发现了自己计算机程序中的一个bug。修正之后，他们的答案变成了 317,206,375。和坎德拉斯的结果完全一致。

这件事在数学界引起了轩然大波。物理学家用一个尚未被证明的"物理直觉"，不但解决了数学家用传统工具花了一个多世纪才推进了两步的问题，而且一口气给出了所有次数的答案。而严谨的数学家反倒是算错了。

这大概是弦论和卡拉比—丘空间对纯数学最直接、最戏剧性的贡献。

值得一提的是，坎德拉斯本人正是1985年四人论文的第一作者。从1985年卡拉比—丘空间登上弦论舞台，到1991年镜像对称震惊数学界，坎德拉斯和卡拉比—丘空间的缘分贯穿始终。

## 3.3 数学家的回应

物理学家用镜像对称解决了数学问题，数学家们当然不会坐视不管。接下来的十几年里，一系列数学理论被提出来，试图给镜像对称以严格的数学基础。

首先是格罗莫夫—威腾不变量（Gromov-Witten invariants）。俄国数学家格罗莫夫和物理学家威腾各自的工作，为曲线计数问题提供了新的数学框架。1996年，吉文塔尔（Givental）用这个框架严格证明了坎德拉斯公式的正确性。1997年，连文豪、刘克峰和丘成桐也独立地给出了证明。

1994年，俄国数学家孔采维奇（Kontsevich）在苏黎世国际数学家大会上提出了同调镜像对称猜想。这个猜想极为抽象，远远超出了大部分物理学家的数学功底，但它试图给出镜像对称的数学本质。

1996年，斯特鲁明格、丘成桐和扎斯洛（Zaslow）提出了SYZ猜想。SYZ猜想给了镜像对称一个几何解释：镜像流形可以通过对每根纤维做T对偶得到。T对偶是弦论里的一个基本对称性：弦在半径 $R$ 的圆上缠绕，和弦在半径 $1/R$ 的圆上缠绕，物理上等价。SYZ猜想把镜像对称还原到了弦论自己的语言里。

丘成桐在这些工作中扮演了重要角色。他自己参与了SYZ猜想和连—刘—丘定理，还组织推动了数学家理解镜像对称的一系列工作。

回过头来看这段历史，我们可以画出一个漂亮的闭环：

> 数学（卡拉比—丘成桐定理，丘成桐1976年）→ 物理（弦论紧致化，坎德拉斯等1985年）→ 数学（镜像对称，曲线计数，坎德拉斯等1991年）→ 数学（SYZ猜想，1996年；同调镜像对称，孔采维奇1994年）

数学和物理之间这种互相喂养的关系，在科学史上并不罕见（最经典的例子是黎曼几何和广义相对论），但是在如此短的时间内完成如此密集的互动，卡拉比—丘成桐空间这个案例堪称典范。

# 四、第二次超弦革命与M理论

## 4.1 五种弦论的困局

中篇结尾留下了一个悬案：弦论一共有五种。

- I型弦论（Type I）：包含开弦和闭弦，规范群为 $SO(32)$
- IIA型弦论（Type IIA）：只有闭弦，无手性
- IIB型弦论（Type IIB）：只有闭弦，有手性
- 杂化弦论 $SO(32)$（Heterotic $SO(32)$）
- 杂化弦论 $E_8 \times E_8$（Heterotic $E_8 \times E_8$）

为什么会有五种？因为弦有很多种"存在方式"。弦可以是开的（两个端点），也可以是闭的（首尾相连成环）。弦的振动分为左行和右行两个方向，超对称可以只施加在一个方向上，也可以两个方向都施加。不同的选择组合，就像同一套积木按不同规则拼，拼出了五种不同的城堡。

五种弦论都活在10维时空里，都包含引力子，都声称自己是候选的终极理论。但它们给出的低能物理是不同的，因为它们在卡拉比—丘空间上的紧致化方案不同，从而给出不同的粒子谱和相互作用。

如果弦论真的是终极理论，它应该只有一种。五个人都声称自己是真正的皇帝，至少有四个是假的。

这个困局从1985年第一次超弦革命之后就一直悬而未决。物理学家们隐隐约约地感觉到五种弦论之间存在着某种深层联系，但谁也说不清楚。

## 4.2 威腾的石破天惊

1995年3月，南加州大学（USC）主办了Strings '95会议。威腾在会上做了一个报告，提出了一个石破天惊的想法：

> 五种弦论，都是同一个11维理论在不同极限下的近似。

这个11维理论，威腾称之为M理论。

据在场的人回忆，威腾的报告结束后，会场里的气氛可以用"电击"来形容。几百个弦论家一下子意识到，过去十年里各自研究的那五种弦论，原来是同一个东西的五个侧面。会后几天之内，大量物理学家开始重新审视自己的研究方向，试图把自己的工作放进M理论的框架里。

M理论的"M"究竟代表什么？威腾本人也说不清楚。Mystery（神秘），Membrane（膜），Mother（母亲），Magic（魔法），Matrix（矩阵），他在不同场合给过不同的解释。也许"M"代表的就是"我们还不知道它到底是什么"。

这不是一句玩笑话。M理论至今没有完整的数学表述。威顿给出的是一张"地图"：五种弦论和11维超引力是这张地图上的六个"角落"，每一个角落都有严格的数学定义，但地图的中心区域是空白的。M理论的完整定义，30年后仍然是一个开放问题。

尽管如此，M理论的基本图景是清楚的：五种弦论是同一个更深层理论的不同极限。就像一块钻石在不同角度的光线下呈现出不同的色彩。五种弦论之间通过"S对偶"和"T对偶"两种对称性联系在一起，而这张网的中心，就是M理论。

## 4.3 从卡拉比—丘空间到 $G_2$ 流形

对于卡拉比—丘空间来说，M理论带来了一个重要的变化。

弦论是10维的，紧致化需要6个额外维度，所以用的是六实维（复三维）的卡拉比—丘流形。M理论是11维的，紧致化需要7个额外维度。

7维的卡拉比—丘空间？不行。卡拉比—丘空间必须是偶数维的（因为它是复流形），7是奇数，用不了。

取而代之的，是一种叫做 $G_2$ 流形的七维空间。$G_2$ 流形和卡拉比—丘空间一样，具有特殊的对称性结构，能保证紧致化之后保留超对称。可以简单理解为：$G_2$ 流形是卡拉比—丘空间的"11维版本"。

在M理论的某些极限下，$G_2$ 流形退化成卡拉比—丘空间加一个圆，从而回到了弦论的框架。

## 4.4 D膜：弦论里的"墙壁"

第二次超弦革命的另一个重要产物是D膜（D-branes）的发现。

1995年，就在威腾提出M理论的同一年，美国物理学家波尔钦斯基（Joseph Polchinski）发表了一篇论文。他指出，弦论里存在一类高维的延展物体，弦的端点可以附着在上面，就像绳子的一端钉在墙上一样。这些物体被称为D膜（D-branes）。

D膜可以是各种维度的。D0膜是一个点，D1膜是一根线，D2膜是一张面，一直到D9膜。

关键在于：D膜可以包裹在卡拉比—丘空间的各种子流形上。一个D膜包裹在一个2维面上，就产生了四维时空里的一个有效粒子。包裹在不同的子流形上，就产生不同的粒子。

卡拉比—丘空间的几何，再次成为决定物理的关键。不仅空间的整体形状决定了粒子的种类，D膜的"缠绕方式"还进一步丰富了结构。

D膜还带来了一个大胆的图景，叫做"膜世界"（braneworld）：也许我们整个可观测宇宙，就是一张D3膜。

什么意思呢？想象一下，三维空间里有一张二维的薄膜。膜上的蚂蚁只能沿着膜的表面爬行，它感知不到膜外面的第三个维度。在膜世界的图景里，我们就是那只蚂蚁。我们所有的粒子，电子、夸克、光子，都是被"困"在D3膜上的开弦的振动模式，无法离开这张膜。

但引力不一样。引力对应的是闭弦，闭弦没有端点，不需要附着在D膜上，所以引力可以"泄漏"到膜外的高维空间里去。

这就解释了一个古老的谜题：为什么引力比其它三种力弱了那么多？电磁力比引力强大约 $10^{36}$ 倍。标准模型对此没有解释。但在膜世界的图景里，引力其实并不弱，只是"漏"到了额外维度里，被稀释了。就像水管漏水，水压没变，只是水跑到别的地方去了。

这个图景非常有画面感，也是弦论和D膜对普通人最容易理解的一个预言。但遗憾的是，膜世界模型在LHC上同样没有找到任何实验证据。

D膜还带来了另一个副作用：它让弦论的"真空"变得更多了。D膜可以缠绕在卡拉比—丘空间的不同子流形上，每一种缠绕方式对应一种不同的真空。这个副作用，在第五章的"景观问题"里会充分显现。

## 4.5 没有实验的革命

第二次超弦革命确实令人兴奋：五种弦论被统一了，D膜被发现了，M理论的图景被提出了。但是有一个尴尬的事实始终悬在头上：

这一切，没有任何实验证据。

中篇讲过，1984年是粒子物理实验时代的分水岭。在那之后，除了顶夸克（1995）和希格斯粒子（2013），粒子实验几乎没有什么"拿得出手"的新发现了。弦论所预言的超对称粒子，至今没有在LHC上被找到。

更深层的问题是能标差距。弦论的特征能标在普朗克尺度，大约 $10^{19}$ GeV。而LHC能达到的最高能量，约 $10^4$ GeV。两者之间差了15个数量级。

这相当于什么？相当于你用肉眼观察一座山，想看清山上一个原子的结构。这个差距已经大到没有意义的程度。

额外维度呢？如果弦论说的对，我们的世界确实有6个蜷缩的额外维度。但是这些维度被压缩到了普朗克长度（约 $10^{-35}$ 米）的尺度。目前人类能探测的最小尺度约 $10^{-19}$ 米，差了16个数量级。

M理论的情况更窘迫。威腾在1995年提出M理论的时候，连完整的数学表述都没有给出。到今天，30年过去了，M理论的完整定义仍然是一个开放问题。一个连数学形式都没有写清楚的理论，更遑论实验验证了。

这并不意味着弦论"错了"。弦论可能是对的，只是我们目前没有手段去验证。但是一个无法被实验检验的理论，在科学方法论上处于一个非常微妙的位置。这个微妙的位置，在接下来的景观问题里会变得更加尖锐。

# 五、景观问题：卡拉比—丘空间太多了

## 5.1 卡拉比—丘空间有多少种？

丘成桐早在1985年就指出，卡拉比—丘空间具有庞大的数量，远非唯一。当时物理学家们问他：六维的卡拉比—丘空间是不是只有一种？丘成桐遗憾地告诉他们，不是。卡拉比—丘空间的数量即使不是无限多，也是相当庞大的。

这个问题到底有多严重，要等到数学家们认真做分类才清楚。

2000年，维也纳工业大学的数学家克鲁泽（Maximilian Kreuzer）和斯卡克（Harald Skarke）完成了一项浩大的分类工作。他们在四维空间里穷举了所有的"反射多胞形"（reflexive polytopes），一共找到了 4.738亿个（确切数字是 473,800,776）。每一个反射多胞形都可以用来构造卡拉比—丘三维流形。这些卡拉比—丘流形的霍奇数 $(h^{1,1}, h^{2,1})$ 共有 30,108 对不同的取值。

这3万多对霍奇数可以画成一张散点图，被称为霍奇数图（Hodge plot）。这张图长什么样呢？它有一个非常显眼的特征：图形关于 $h^{1,1} = h^{2,1}$ 这条对角线，呈现出近似的镜像对称。也就是说，如果存在一个霍奇数为 $(a, b)$ 的卡拉比—丘空间，那么很可能也存在一个霍奇数为 $(b, a)$ 的卡拉比—丘空间。

这正是上一章讲的镜像对称在几何分类上的体现。镜像对称不再只是一个物理直觉，它直接写在了卡拉比—丘空间的分类表里。

需要注意的是，4.738亿个反射多胞形并不等于4.738亿种卡拉比—丘空间。一个多胞形可以有多种不同的"三角剖分"（triangulation），每种剖分给出一个不同的卡拉比—丘流形。所以真正的卡拉比—丘三维流形的数量，远远大于4.738亿。确切有多少种拓扑不同的卡拉比—丘三维流形，至今仍然是一个开放的数学问题。

## 5.2 弦论景观

卡拉比—丘空间数量庞大这件事本身还不是最可怕的。最可怕的是，当你把D膜、磁通量（flux）以及其它所有的选择都考虑进来之后，弦论可能的"真空态"的数目变得天文数字级别地大。

2000年，布索（Bousso）和波尔钦斯基首先论证了弦论可以产生大量的真空。2003年，四位物理学家提出了所谓的KKLT机制，给出了一种具体的真空构造方案。在这个方案的基础上，道格拉斯（Michael Douglas）估算了弦论真空的总数目。

结果是：超过 $10^{500}$。

这就是所谓的弦论景观（String Landscape）。

$10^{500}$ 是什么概念？可观测宇宙中的原子总数大约是 $10^{80}$。$10^{500}$ 比 $10^{80}$ 大了 $10^{420}$ 倍。这不是一个"很大"的数字，这是一个大到人类语言都无法有效表达的数字。

每一个真空对应一种不同的低能物理：不同的粒子质量、不同的耦合常数、不同的宇宙常数。换句话说，弦论预言了 $10^{500}$ 种可能的宇宙，而不是唯一的一种。

这和物理学家们最初的梦想背道而驰。他们希望弦论是一种"终极理论"，能够唯一地预言我们宇宙的所有参数。现在弦论告诉他们：我能解释一切，但解释得太多了，我解释了 $10^{500}$ 种可能的宇宙，至于你活在哪一种里，我说不清楚。

## 5.3 人择原理与多重宇宙

面对景观问题，物理学家们分裂成了两派。

一派以萨斯坎德（Leonard Susskind）为代表，拥抱景观。他们认为，$10^{500}$ 种真空都是真实存在的，对应 $10^{500}$ 个平行宇宙。我们活在这个宇宙，是因为这个卡拉比—丘空间的几何"恰好"允许生命存在：$|\chi|/2 = 3$，恰好三代费米子，恰好可以形成稳定的原子和化学。

这就是人择原理（Anthropic Principle）：我们观测到的宇宙参数之所以长成这样，是因为只有这样的参数组合才允许"我们"存在并观测。物理定律本身并没有要求它必须如此。

萨斯坎德说，景观不是弦论的缺陷，而是弦论的特征。宇宙的"精调"不需要上帝之手，只需要足够多的宇宙。

另一派认为，如果弦论无法预言唯一的真空，那弦论就不是一个真正的物理理论，而只是一个数学框架。

这一派的代表人物包括理论物理学家斯莫林（Lee Smolin）和数学家沃伊特（Peter Woit）。2006年，两人几乎同时出版了批评弦论的著作。斯莫林的《物理学的困惑》（*The Trouble with Physics*）和沃伊特的《连错都算不上》（*Not Even Wrong*）都对弦论提出了严厉的批评。

他们的核心论点是：一个无法做出可被实验检验的独特预言的理论，在科学方法论的意义上已经不再是物理学了。斯莫林还批评说，弦论在美国理论物理学界形成了不健康的垄断地位，排挤了其它研究方向。

沃伊特的书名来自泡利的名言："它甚至连错都算不上"（It is not even wrong）。意思是论点根本无法被证伪，连"错误"的资格都没有。沃伊特用这句话形容弦论：预言太多太泛，无法被实验证伪。

上一节我们已经看到了那个15个数量级的能标差距。面对这个差距，弦论的支持者和反对者给出了截然不同的态度：支持者说"我们需要更好的实验"，反对者说"一个永远推到'更高能标'的理论不是科学"。

这场争论至今没有定论。

## 5.4 丘成桐的立场

作为卡拉比—丘空间的数学创造者，丘成桐对景观问题和人择原理持什么态度呢？

丘成桐的立场比较独特。他既不拥抱景观，也不全盘否定弦论。

在《大宇之形》中，丘成桐说（大意）：他没有真正参与这场争论，这也许是作为数学家的一种奢侈。他不需要为威胁着物理学界的东西而心烦意乱，而是可以坐在一旁问自己一贯的问题：数学家能为这个困境提供什么样的帮助？

丘成桐认为，卡拉比—丘空间的分类是一个严肃的数学问题，物理学家不应该轻易用人择原理来绕过它。如果卡拉比—丘空间太多了，也许意味着我们的数学理解还不够深。真正的出路，在于从数学结构里找到进一步限制真空的原则。

这个立场很"丘成桐"。他一辈子的工作方式就是：物理学家提出问题，数学家回答问题。卡拉比猜想如此，卡拉比—丘空间的弦论应用如此，镜像对称的数学基础如此，景观问题也是如此。

## 5.5 卡拉比—丘空间的物理后果：一个讽刺

回过头来看，景观问题的根源，正是卡拉比—丘空间的多样性。

1976年，丘成桐证明了卡拉比猜想，给出了一大类满足特定条件的空间的存在性。这个定理在纯数学上是漂亮的，因为它告诉我们这类空间是"丰富的"。但正是这种丰富性，在1985年被弦论借用之后，变成了物理上的一个大麻烦。

卡拉比—丘空间越多，弦论的预言能力就越弱。

这是丘成桐1976年的数学定理带来的一个意想不到的物理后果。数学上的丰饶（richness），变成了物理上的尴尬（embarrassment）。这大概是科学史上最具讽刺性的故事之一。

# 六、弦论凉了吗？

## 6.1 三重打击

弦论在1980年代中期和1990年代中期经历了两次"革命"，红火了十几年。但到了2000年代中后期，热度明显退了下来。到今天，弦论不能说"死了"，但说它"凉了"并不过分。

凉的原因是三重打击几乎同时到来。

第一重打击：实验端彻底断供。

中篇讲过，1984年是粒子物理实验时代的分水岭。在那之后，标准模型的最后一块拼图，希格斯粒子，在2013年被LHC找到了。标准模型"完工"了。

但是，LHC花了几百亿欧元，除了确认标准模型是对的之外，没有发现任何超越标准模型的新粒子。超对称粒子没有。额外维度的信号没有。微型黑洞没有。弦论预言的一切"新物理"，在LHC上全部缺席。

更关键的是，这已经超过"还没找到"的范畴，在可预见的技术条件下根本不可能找到。弦论的特征能标在普朗克尺度（$10^{19}$ GeV），LHC只能到 $10^4$ GeV。要建造一台能达到普朗克能标的加速器，按照现有技术估算，环形周长需要达到银河系的尺度。更多经费也解决不了，这在物理上不可能。

标准模型之上，实验能触及的最后一个前沿是中微子物理。中微子有质量（这已被实验证实），而标准模型里中微子没有质量，所以中微子质量是超越标准模型的确切证据。但中微子质量的物理来源有很多种可能的解释，弦论只是其中之一，而且不是最简洁的那个。

第二重打击：理论端自己崩了。

景观问题从根本上动摇了弦论作为"终极理论"的资格，已经超出"还需要研究"的范畴。

$10^{500}$ 种真空意味着弦论什么都能解释。什么都能解释，就等于什么都没解释。物理学的核心标准是可证伪性（波普尔的科学哲学），一个无法排除任何可能性的理论，在科学哲学的意义上已经退出了物理学的范畴。

弦论家的辩护是：弦论不是"什么都能解释"，它的 $10^{500}$ 种真空都是有具体数学结构的，只是我们还没找到挑选正确真空的原则。但这种辩护在40年后听起来越来越苍白。

第三重打击：人才在流失。

这是最现实的指标。年轻的理论物理学家越来越少选弦论作为研究方向了。

为什么？原因其实不复杂，归结到底就是三个字：没反馈。

做实验物理的人，设计一个实验，跑几个月或者几年，出数据，发论文。数据要么支持你的假说，要么否定它。无论哪种结果，你都"知道了一些东西"。做凝聚态理论的人，预言一种新材料的性质，实验组几个月后就能合成出来验证。做机器学习的人，写一个模型，跑一下benchmark，数字好不好一目了然。

做弦论的人呢？你花三年推导了一个卡拉比—丘空间上的紧致化方案，算出了一组粒子谱。然后呢？没有任何实验可以告诉你这个方案对不对。你永远不知道自己是在正确的路上走了一步，还是在错误的路上走了一步。

这种"没有反馈"的状态，对一个研究者的心理消耗是巨大的。物理学的传统是理论和实验的对话。牛顿预言了苹果落地的轨迹，你可以扔一个苹果验证。爱因斯坦预言了光线弯曲，爱丁顿去非洲拍了日食照片。杨振宁预言了规范玻色子，实验组在加速器里找到了。这种对话是物理学的生命力所在。弦论把这种对话切断了。

2000年代，弦论还是理论物理的"显学"，美国顶尖大学物理系的理论组几乎都以弦论为主。到了2020年代，情况已经明显不同。凝聚态物理、量子信息、量子计算、AI for Science，这些有实验反馈、有工业界资金支持的方向在大量吸走人才。

除了学术上"没反馈"，还有一个很现实的问题：就业。做弦论的博士毕业之后，学术职位越来越少（理论物理的终身教职本来就少，弦论方向更少），工业界也不需要弦论的技能。相比之下，做量子计算的博士可以去Google、IBM，做机器学习的博士可以去任何一家科技公司。一个聪明的22岁年轻人，选方向的时候不可能不考虑这些。

一位弦论家曾经感慨：弦论最大的问题不是理论上的，而是你没法告诉一个聪明的博士生，"你花五年研究弦论，毕业之后有什么具体的问题可以解决"。

这不是弦论的"错"，但这是弦论的现实。

## 6.2 弦论变成了数学？

弦论凉了，但它并没有消失。它在以一种不同的方式继续存在着。

很多当年做弦论的物理学家，现在实际上在做的是数学。镜像对称、同调代数、格罗莫夫—威腾不变量，这些从弦论里生长出来的数学工具，已经成为纯数学的活跃研究方向。弦论作为物理理论也许"凉了"，但弦论作为数学的灵感来源，依然是热的。

这是一个有趣的历史轮回。丘成桐在1976年证明了一个纯数学定理，1985年这个定理被弦论借走了，物理学家们用它来追求终极理论。40年后，终极理论没有实现，但弦论留下的数学遗产是真实的。镜像对称反哺了计数几何，这不会因为弦论"凉了"而消失。

还有一类工作介于数学和物理之间。AdS/CFT对偶（反德西特空间/共形场论对偶）是弦论在1997年产生的一个成果，由马尔达塞纳（Juan Maldacena）提出。AdS/CFT不依赖于弦论是否是"正确的"终极理论，它本身是一个数学等价关系，目前被广泛应用在量子引力、黑洞信息悖论、甚至凝聚态物理的研究中。

弦论也许最终不是物理学的终极理论，但它可能是人类发展出来的最强大的"理论物理工具箱"之一。

## 6.3 丘成桐的收获

在这场弦论的兴衰起落中，丘成桐大概是最坦然的一个人。

他证明的卡拉比—丘成桐定理，无论弦论最终成功还是失败，作为数学定理永远成立。数学不需要实验验证。$2+2=4$ 不会因为物理学家改变了理论而变成5。

弦论借用了卡拉比—丘空间，镜像对称从弦论里生长出来反哺了数学，SYZ猜想打开了新的几何研究方向。这些数学贡献是实打实的，不会因为弦论的物理地位下降而贬值。

回过头来看，丘成桐当年（1985年）给弦论泼的那盆冷水，说卡拉比—丘空间"不是唯一的，数量很多"，其实是最早的预警。物理学家们当时不以为然，觉得可以解决。40年后，这个问题不但没有解决，反而变成了 $10^{500}$ 种真空的景观灾难。

数学家有时候比物理学家看得更准。因为数学家不需要对实验负责，也不需要对"终极理论"抱有信仰。他们只需要对定理的正确性负责。

2022年，73岁的丘成桐从哈佛大学全职回到中国，出任清华大学求真书院院长。从美国到中国，从研究者到教育者，丘成桐晚年的重心不再是证明新的定理，而是"种树"。

求真书院的模式很简单：从高中阶段就选拔数学天才，用八年时间（本科加博士）把他们培养成一流的数学家。丘成桐亲自设计课程体系，亲自面试学生，甚至亲自给本科生上课。他要在中国土地上复制当年伯克利和哈佛给他的那种数学教育。

这件事和弦论有什么关系？表面上没有。但是往深了想，丘成桐的选择本身就说明了一个道理：数学的价值不依赖于任何一个物理理论的成败。卡拉比—丘成桐定理不需要弦论来"证明"自己的意义。它在纯数学里的地位，从1976年到今天，从来没有动摇过。

1970年，一个22岁的香港年轻人在伯克利的课堂上被广义相对论击中，开始思考空间的曲率。50多年后，这个年轻人回到了东方，坐在清华园里，教另一群年轻人思考同样的问题。

弦论也许凉了。但愿数学不会凉。

# 七、小结

回到上篇的起点。

1970年，在伯克利读博的丘成桐旁听了广义相对论课，提出了一个问题：

> 没有任何能量动量的全局空间，可以是弯曲的吗？

答案是肯定的。卡拉比—丘成桐定理告诉我们，满足第一陈类为零的紧致凯勒流形，必然存在里奇平坦的度规。也就是说，这样的空间在没有任何物质的情况下依然可以是弯曲的。

回到中篇的场景。

1984年，丘成桐凝视着圣地亚哥拉霍亚的太平洋，接到了斯特鲁明格的电话。超弦理论需要卡拉比—丘成桐空间。丘成桐感慨道：难道物理终极理论的大门终于要被开启了？

现在，到了下篇的结尾。几十年过去了，这扇门究竟有没有被开启？

坦率地说，弦论给出了一幅宏大的图景。卡拉比—丘空间的几何决定了粒子种类，欧拉示性数决定了费米子的世代数，镜像对称让弦论反哺了数学，M理论试图统一五种弦论。这幅图景是壮丽的。

但壮丽的图景和现实之间隔着一道鸿沟。15个数量级的能标差距，LHC上始终找不到的超对称粒子，$10^{500}$ 种可能的真空。弦论不是错了，而是对得太泛了。用沃伊特的话说，"连错都算不上"。

也许这扇门还没有真正打开。也许弦论不是最终的答案，只是通往答案的路上的一个路标。也许需要更深入的数学理解，才能把 $10^{500}$ 种可能压缩到寥寥几种，甚至唯一的一种。

整个三篇长文，走过了这样一条路线：

- 上篇（数学）：从爱因斯坦方程出发，经过流形、凯勒流形、陈省身类、欧拉示性数，到达卡拉比猜想和丘成桐的证明。
- 中篇（物理史）：从柏拉图的第一性原理出发，经过量子力学、规范场论、标准模型、超对称，到达弦论和1985年卡拉比—丘空间登上弦论舞台。
- 下篇（物理应用与困境）：从霍奇数和费米子代数出发，经过镜像对称、M理论、D膜，再到实验的缺席、景观问题和弦论的降温。

三篇合起来，就是一个完整的故事：一个博士生1970年在课堂上的直觉，如何变成一个数学定理，又如何变成物理学终极理论的数学基础，最后又如何目睹这个终极理论陷入困境。

400年前，开普勒用五种正多面体来解释太阳系六颗行星的轨道间距。这个模型是错的。但正是这个错误的追求，推动开普勒发现了行星运动三定律，为牛顿力学铺平了道路。错误的终极模型，有时候比没有模型更有价值。

弦论和卡拉比—丘空间的故事，也许就是我们这个时代的"开普勒多面体"。一段美丽的弯路，一笔真实的数学遗产，以及一个尚未回答的问题：宇宙的终极结构，究竟是不是几何的？

_（全文完）_

> ## 一、引言
>
> 回扣中篇结尾的场景：1985年，坎德拉斯、霍洛维茨、斯特鲁明格、威腾四人论文横空出世，卡拉比—丘空间作为弦论多出来的六维登场。物理学家兴奋了——但兴奋之后，一个更深的问题随之而来：
>
> 卡拉比—丘空间究竟在弦论里扮演什么角色？
>
> 它不只是"满足要求的一个空间"。弦论要求的10维时空，4维是我们熟悉的相对论时空，另外6维蜷缩在卡拉比—丘空间里。这意味着，我们观测到的一切低能物理——粒子的种类、质量、相互作用——都由这个蜷缩的卡拉比—丘空间的几何形状决定。
>
> 卡拉比—丘空间，是弦论和现实物理世界之间的翻译器。
>
> 本篇的任务，就是把这个翻译的过程说清楚。
>

> ## 二、从几何到物理：卡拉比—丘空间决定了什么
>
> ### 2.1 霍奇数
>
> 卡拉比—丘空间的几何，可以用一组数来描述，这组数叫做霍奇数（Hodge numbers），记为 $h^{p,q}$。
>
> 对于复三维的卡拉比—丘流形，最重要的两个霍奇数是：
>
> - $h^{1,1}$：对应卡拉比—丘空间里独立的"膨胀模"（Kähler模），决定了低能物理里某类无质量场（轴子场）的数目
> - $h^{2,1}$：对应卡拉比—丘空间里独立的"形变模"，决定了另一类无质量场（模量场）的数目
>
> 【需要介绍霍奇数的直观含义：说的是卡拉比—丘空间里有多少种"独立的形状变化方式"，类似于一个多面体有多少种独立的变形自由度】
>
> ### 2.2 欧拉示性数与粒子代数
>
> 回扣上篇第五章的数学铺垫——欧拉示性数 $\chi$。
>
> 上篇我们说过，欧拉示性数是拓扑学的基石，刻画了空间的整体拓扑性质。对于卡拉比—丘三维流形：
>
> $$\chi = 2(h^{1,1} - h^{2,1})$$
>
> 这个公式把上篇的数学（欧拉示性数）和霍奇数直接联系了起来。
>
> 更重要的是，$|\chi|/2$ 对应弦论低能极限里费米子的代数，也就是我们观测到的夸克和轻子的"世代数"（generation number）。
>
> 我们知道，标准模型里的费米子有三代：
>
> - 第一代：上夸克、下夸克、电子、电子中微子
> - 第二代：粲夸克、奇夸克、渺子、渺子中微子
> - 第三代：顶夸克、底夸克、陶子、陶子中微子
>
> 为什么恰好是三代？标准模型对此没有任何解释，三代费米子是实验给出的既成事实。
>
> 而弦论给出了一个几何上的解释：如果 $|\chi|/2 = 3$，即卡拉比—丘空间的欧拉示性数为 $\pm 6$，则弦论预测恰好有三代费米子。
>
> 这是整篇文章最关键的一句话，也是上篇数学和下篇物理之间最直接的一次握手：
>
> > 陈省身—高斯—博内公式里的欧拉示性数，决定了我们宇宙里费米子的世代数。
>
> 【需要展开说明这个对应关系的来龙去脉，以及丘成桐对此的反应】
>
> ### 2.3 一句话总结
>
> 卡拉比—丘空间的几何数据（霍奇数、欧拉示性数）和低能物理的对应关系，可以粗略地总结如下：
>
> |卡拉比—丘空间的几何|低能物理|
> |---|---|
> |$h^{1,1}$|某类无质量标量场的数目|
> |$h^{2,1}$|另一类无质量标量场的数目|
> |$\|\chi\|/2 = \|h^{1,1} - h^{2,1}\|$|费米子世代数|
> |$SU(3)$和乐群|保留 $\mathcal{N}=1$ 超对称|
>
> 卡拉比—丘空间的形状，决定了我们看到的粒子世界。
>

> ## 三、镜像对称：弦论反哺数学
>
> ### 3.1 镜像对称的发现
>
> 1989年前后，物理学家们在研究弦论在卡拉比—丘空间上的紧致化时，发现了一个令人惊讶的现象：
>
> > 两个拓扑结构完全不同的卡拉比—丘空间，可以给出完全相同的弦论物理。
>
> 具体地说，如果一个卡拉比—丘空间的霍奇数是 $(h^{1,1}, h^{2,1})$，那么存在另一个卡拉比—丘空间，其霍奇数是 $(h^{2,1}, h^{1,1})$——也就是两个霍奇数互换——而这两个空间给出完全相同的弦论物理。
>
> 这两个卡拉比—丘空间被称为互为镜像（mirror）。
>
> 【需要解释为什么这件事令人惊讶：两个拓扑上完全不同的空间——欧拉示性数甚至符号相反——在物理上却完全等价。这在纯数学里是没有预兆的。】
>
> ### 3.2 坎德拉斯的计算：震惊数学界
>
> 镜像对称最惊人的应用，发生在1991年。
>
> 数学上有一个古老的问题：一个五次曲面（quintic threefold）上有多少条给定次数的有理曲线？
>
> 这是代数几何里的经典问题。对于次数为1（直线）的情况，答案是2875条，19世纪就算出来了。对于次数为2的情况，数学家费了很大力气，在1986年算出了609250条。对于次数为3，数学家正在努力计算中。
>
> 1991年，物理学家坎德拉斯（Candelas）用镜像对称，把这个问题转化到镜像卡拉比—丘空间上的一个等价计算，一口气给出了所有次数的答案，其中次数为3的答案是317206375。
>
> 数学家们用传统方法核验了次数为3的结果，答案一致。
>
> 这件事在数学界引起了轩然大波。物理学家用一个"物理直觉"——镜像对称——解决了数学家用传统工具无法解决的问题。这是弦论和卡拉比—丘空间对纯数学最直接、最戏剧性的贡献。
>
> 【需要说明：坎德拉斯本人正是1985年四人论文的作者之一，他和卡拉比—丘空间的缘分贯穿始终】
>
> ### 3.3 数学家的回应
>
> 镜像对称的数学基础，随后由数学家们补全。
>
> 丘成桐在其中扮演了重要角色，他组织和推动了数学家理解镜像对称的工作。后来，格罗莫夫—威腾不变量、同调镜像对称猜想（Kontsevich，1994年）等一系列数学理论相继提出，试图给镜像对称以严格的数学基础。
>
> 这是整个系列最漂亮的闭环：
>
> > 数学（CY定理，丘成桐1976年）→ 物理（弦论紧致化，1985年）→ 数学（镜像对称，曲线计数，1991年）
>

> ## 四、第二次超弦革命与M理论
>
> ### 4.1 五种弦论的困局
>
> 中篇留下了第一个悬案：弦论不是一种，而是五种：
>
> - I型弦论（开弦+闭弦，$SO(32)$）
> - IIA型弦论（闭弦，无手性）
> - IIB型弦论（闭弦，有手性）
> - 杂化弦论 $SO(32)$
> - 杂化弦论 $E_8 \times E_8$
>
> 五种弦论都声称是终极理论，究竟哪种才是"真正的"弦论？
>
> 【以卡拉比—丘空间为主线来讲：这五种弦论对应五种不同的紧致化方案，而不同的紧致化方案选用不同类型的卡拉比—丘空间，给出不同的低能物理。这不是五个理论，而像是同一个大理论的五个不同"侧面"。】
>
> ### 4.2 威腾的M理论
>
> 1995年，威腾（Edward Witten）在一次弦论会议上提出了震惊物理界的想法：
>
> > 五种弦论，都是同一个11维理论在不同极限下的近似。这个11维理论，威腾称之为M理论。
>
> M理论的"M"究竟代表什么，威腾本人也说不清楚（Mystery？Membrane？Mother？）。
>
> M理论的空间维度是11维：10维时空再加1维。多出来的这一维，在不同的极限下收缩到不同的大小，从而给出五种不同的弦论。
>
> 【以卡拉比—丘空间为主线：M理论的紧致化不再用卡拉比—丘三维流形（六实维），而是用$G_2$流形（七实维）。$G_2$流形是卡拉比—丘空间的"11维版本"，同样具有特殊的和乐群结构。卡拉比—丘空间在M理论里是$G_2$流形的一个特殊极限。】
>
> ### 4.3 D膜的出现与卡拉比—丘空间
>
> 第二次超弦革命的另一个重要产物是D膜（D-branes）的发现（波尔钦斯基，1995年）。
>
> D膜是弦论里的高维延展物体，弦的端点可以附着在D膜上。D膜可以包裹在卡拉比—丘空间的各种子流形上，由此产生丰富的物理效应。
>
> 【点明卡拉比—丘空间的作用：D膜包裹卡拉比—丘空间的哪些子流形，决定了低能物理里有哪些额外的规范群和物质场。卡拉比—丘空间的几何，再次成为决定物理的关键。】
>

> ## 五、景观问题：卡拉比—丘空间太多了
>
> ### 5.1 卡拉比—丘空间的分类
>
> 丘成桐早在1985年就指出，卡拉比—丘空间不是唯一的，而是具有庞大的数量。
>
> 问题有多严重？
>
> 数学家和物理学家们花了几十年时间，试图对卡拉比—丘空间进行分类。目前已知的复三维卡拉比—丘流形，其霍奇数 $(h^{1,1}, h^{2,1})$ 的分布可以画成一张图，被称为霍奇数图（Hodge plot）。这张图上有数万个点，每个点对应一种卡拉比—丘空间，而每种卡拉比—丘空间对应一种可能的弦论真空，一种可能的宇宙。
>
> 【可以在这里描述霍奇数图的形状：它关于 $h^{1,1} = h^{2,1}$ 这条线具有镜像对称，这正是镜像对称的几何体现。】
>
> ### 5.2 弦论景观
>
> 把所有可能的卡拉比—丘空间、所有可能的D膜配置、所有可能的磁通量（flux）都考虑进来，弦论的真空数目估计超过 $10^{500}$。
>
> 这就是所谓的弦论景观（String Landscape）。
>
> 每一个真空对应一种不同的低能物理：不同的粒子质量、不同的耦合常数、不同的宇宙常数。换句话说，弦论预言了 $10^{500}$ 种可能的宇宙，而不是唯一的一种。
>
> ### 5.3 人择原理与多重宇宙
>
> 面对景观问题，物理学家们分裂成了两派：
>
> 一派认为，这 $10^{500}$ 种真空都是真实存在的，对应 $10^{500}$ 个平行宇宙。我们活在这个宇宙，是因为这个卡拉比—丘空间的几何"恰好"允许生命存在（比如 $|\chi|/2 = 3$，恰好三代费米子，恰好可以形成稳定的原子和化学）。这就是人择原理（Anthropic Principle）与多重宇宙的结合。
>
> 另一派认为，如果弦论无法预言唯一的真空，那弦论就不是一个真正的物理理论，而只是一个数学框架。斯莫林（Lee Smolin）和沃伊特（Peter Woit）等人对弦论提出了严厉批评。
>
> 【点明卡拉比—丘空间在这个争议里的核心地位：景观问题的根源，正是卡拉比—丘空间的多样性。卡拉比—丘空间越多，弦论的预言能力就越弱。这是丘成桐1976年的数学定理带来的一个意想不到的物理后果。】
>
> ### 5.4 丘成桐的立场
>
> 丘成桐对景观问题和人择原理的态度比较保守。他认为，卡拉比—丘空间的分类问题是一个严肃的数学问题，物理学家不应该轻易用人择原理来绕过它。
>
> 真正的出路，在于更深入地理解卡拉比—丘空间的数学结构，从中找到进一步限制真空的原则。
>

> ## 六、小结
>
> 回到上篇的起点：1970年，在伯克利读博的丘成桐旁听了广义相对论课，提出了一个问题：
>
> > 没有任何能量动量的全局空间，可以是弯曲的吗？
>
> 答案是肯定的。卡拉比—丘成桐定理告诉我们，满足第一陈类为零的紧致凯勒流形，必然是里奇平坦的，也就是说，这样的空间在没有任何物质的情况下依然可以是弯曲的。
>
> 回到中篇的场景：1984年，丘成桐凝视着圣地亚哥拉霍亚的太平洋，接到了斯特鲁明格的电话，感慨道：难道物理终极理论的大门终于要被开启了？
>
> 几十年过去了，这扇门究竟有没有被开启？
>
> 弦论给出了一幅宏大的图景：10维时空，其中6维蜷缩在卡拉比—丘空间里，卡拉比—丘空间的几何决定了我们宇宙里的粒子种类和相互作用。镜像对称让弦论反哺了数学，M理论试图统一五种弦论，景观问题揭示了这幅图景的复杂性。
>
> 但景观问题也留下了一个深刻的困境：如果卡拉比—丘空间有 $10^{500}$ 种，弦论的预言能力从何而来？
>
> 也许，问题的答案还藏在卡拉比—丘空间的数学里，等待着下一个丘成桐。
>
