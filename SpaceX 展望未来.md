# 引子

2026年6月12日，纳斯达克。SpaceX以代码SPCX挂牌，发行价135美元，募资约750亿美元，对应整体估值约1.77万亿。这是人类有史以来募资额最大的一次IPO。在它之前，这个世界纪录由沙特阿美（沙特国家石油公司）2019年上市时创下，约290亿美元，SpaceX这次募到的钱接近它的三倍。首日收盘160.95美元，涨约19%。

数字很大，但真正值得停下来看的是两个小数字。一个是，这次卖出去的只占SpaceX全部股权的4.3%。另一个是，上市之后，Elon Musk一个人握着约82.4%的投票权。

也就是说，人类把"通往太空"这件事第一次摆上了公开市场的货架，普通人用一个股票账户就能买进一小片。但你买到的基本只是分一份涨跌：股价涨你赚、跌你亏，至于这家公司往哪走、做什么决定，话语权仍然牢牢握在Musk一个人手里。

这篇文章想理清的就是这件事的来龙去脉，以及它的赌注：太空进入权怎么从国家落到一家公司手里，这家公司到底是什么构造，市场凭什么给它标出1.77万亿，它的账本里藏着什么，还有当一个文明级的基础设施被一个人攥着，我们该怎么看。

# 一、从举国之力到一个股票代码

要掂量6月12日的分量，得先记起太空原本是什么。

## 1.1 太空曾经只属于国家

二十世纪的航天，是国家意志的直接延伸。把人送上月球，需要一个国家把数十万人、几个百分点的GDP、加上整整一代人的自尊，押在同一个方向上。这是文明在向宇宙宣示存在，谈不上是一桩生意。

太空因此天然带着一层神圣感。它太贵、太险、太远，落不到任何个人头上，只能属于"我们所有人"，通过"国家"这个唯一负担得起它的容器。在那半个世纪里，"上太空"和"买股票"是两个不会出现在同一句话里的词。

## 1.2 国家退场，私人接手

转折发生得很安静。登月的政治激情退潮后，航天预算被反复削减，载人航天慢慢变成一项昂贵、缓慢、风险极高的遗产工程。国家没有离开太空，但它开始把越来越多的活儿外包出去：发射、补给，后来连载人也外包。

钻进这个缝隙的，是一家2002年成立的小公司。它靠创始人卖掉支付公司换来的约一亿美元起步，头三次发射全部失败，差点关门。它用一件最朴素也最颠覆的事改写了规则：让火箭能重复使用。

当一枚火箭能像飞机一样落回来、再飞一次，把东西送上轨道的成本就被砍掉了一个数量级。我自己跟着这个行业看了这些年，最意外的是这道"贵到只有国家玩得起"的护城河，塌得比几乎所有人预想的都快。

## 1.3 钥匙易主

到今天，这家公司已经不只是参与航天，而是握住了通往轨道的钥匙。它是目前唯一能把宇航员稳定送上空间站、再接回来的美国力量，发射频次高到对手望尘莫及，连国家级航天机构在不少关键任务上都离不开它。

这就是我们走到6月12日的全部前提：太空进入权这件曾经只有超级大国掌握的东西，已经实质性地落进了一家私人公司手里。而那一天，这家公司又把自己的一小片所有权摆上了公开市场。

从举国之力到一个股票代码，人类用了半个多世纪。

# 二、这是一台什么样的机器

很多人到今天还以为SPCX是"那家造火箭的公司"。这是个不小的低估。

## 2.1 四条命脉

今天挂在纳斯达克上的这个实体，是一套罕见的、端到端的基础设施栈。它握着四样东西。

火箭与飞船，是进出太空的钥匙，谁掌握它谁就守着通往一切太空生意的收费站。Starlink，是地球的通信底层，一张铺在近地轨道上、覆盖全球的卫星互联网，正在变成像水电一样的底座。xAI，是智能层，2026年2月Musk把这块AI业务连同社交平台X一起并进了SpaceX。芯片与机器人，是伸向物理世界的手脚，从AI芯片产能到具身智能的设想。

把这四样放在一起看，它几乎覆盖了一个未来社会运转所需的基础层：能量与运输、连接、智能、计算。这已经是一套基础设施操作系统的雏形，装不进"业务组合"这个词里。

## 2.2 野心从火星转向月球

值得留意的是，这套野心最近换了挡。那个被讲了很多年的火星梦，正在让位给一个更近、更急的目标：月球。

理由相当务实。去火星要等每26个月一次的行星窗口，去月球每10天就能发射一次。窗口越密，迭代越快，商业兑现就越早。2026年2月，Musk公开把重心转向月球，新版蓝图是在月球上建厂、批量制造AI卫星，再用一台巨型电磁弹射器（他称之为"质量投射器"）把卫星甩上轨道。

火星没有被放弃，Musk给的时间表是五到七年后才正式起步。但这个转向本身就说明了问题：当一个太空梦想需要股票市场买单，连"先去哪颗星球"都会按"哪条路兑现得更快"重新排序。

## 2.3 历史的回声

如果"一家公司握住一国乃至全球命脉"这个画面让你隐隐不安，那是因为它在历史上有过先例。

十七世纪的东印度公司，手里握着军队、铸币权和殖民地，是一个有"准主权"的商业实体。十九世纪的铁路大亨和标准石油，用控股公司层层嵌套，垂直垄断了一个工业国的血管。他们证明过同一件事：当一家私人实体掌握了一个时代最关键的基础设施，"公司"这个词就装不下它真正在做的事了。

SpaceX是这个谱系在二十一世纪的最新一员，只不过它垄断的从香料、铁路、石油，换成了通往太空的路、覆盖地球的网和正在崛起的智能。它的宏大和它的危险，从一开始就是同一件事的两面。

# 三、上市：公众的钱，一个人的方向盘

上市这件事真正特殊的地方，跟股市常识关系不大，在于一句话：钱来自公众，方向只对一个人负责。

## 3.1 募750亿，只卖了4.3%

先回答最朴素的问题：这场史上最大IPO到底卖了什么、卖了多少。

SpaceX新发行约5.556亿股A类普通股，每股135美元，总共募集约750亿美元，对应整体估值约1.77万亿。这个募资额是此前纪录沙特阿美那场（约290亿美元）的三倍，把它远远甩在身后。

更值得看的是背后那个小数字：这次卖出去的只占公司全部股权的4.3%，剩下的九成五牢牢留在Musk和原有股东手里。而且卖的几乎全是新发行的股票（外加约37.5亿美元的"亲友"份额），是公司在增发融资，老股东并没有借机套现离场。超大的融资额配上超小的流通盘，公众能买到、能交易的筹码极少。

## 3.2 双层股权：一个人攥着方向盘

这就要讲到那个对每个买入者都要紧、却最少被普通人注意的设计：双层股权。

SpaceX的股票分两种。公开市场上你能买到的是A类股，1股1票。Musk和内部人持有的是B类股，1股10票。效果是这样的：

| | 股权（经济利益） | 投票权（控制权） |
|---|---|---|
| Elon Musk | 约42% | 约82.4% |
| 公众及其他股东 | 约58% | 约18% |

Musk用不到一半的股权，锁住了超过八成的投票权。这意味着不管你买入多少SPCX，买到的几乎只是经济敞口：你分享它涨跌的收益，却几乎分不到任何"决定它往哪走"的权力。通往太空的方向盘，法律意义上由一个人独自掌握，而且这个人不可能被股东罢免。

到这里，全文真正的张力就显形了。股票市场是一台精密却短视的机器，它每天开盘收盘，每三个月用一次财报检验承诺，它贴现的"未来"通常以季度、最多以年为单位。而SpaceX卖给市场的是火星、月球工厂、几十年后才可能兑现的多行星文明，这是一个以世纪为尺度的故事。一个世纪级的愿景，被装进一台季度级的定价机器，再交给一个不受制衡的人掌舵，这就是买入SPCX时你真正下注的东西。

至于上市后股价为什么忽上忽下，道理和别的热门新股没两样：流通盘小、指数纳入带来的被动买盘把价格往上顶，年底一批锁定股解禁又可能把它压下来，再加上Musk一句话就能搅动情绪。这些机制每只热门新股都有，跟SpaceX的特殊性无关，这里不展开。

## 3.3 估值鸿沟：那一万亿是叙事溢价

撇开短期波动，这家公司长期到底值多少？这件事最迷人的地方在于，严肃机构给出的答案差了一倍还多。

光谱的一端是激进派。方舟投资（ARK Invest）用蒙特卡洛模拟，把17个关键变量（Starlink用户增速、单用户收入、发射成本、AI兑现概率等）各设一个区间，让计算机跑成千上万次随机组合，得出一个概率分布。它的结论是，SpaceX到2030年的企业价值基准约2.5万亿美元，悲观情形1.7万亿，乐观情形3.1万亿。撑起这个数的核心假设，是Starlink到2035年能做到每年约3000亿美元收入、拿下全球通信市场约15%的份额。

光谱的另一端是保守派。以严谨著称的Morningstar给SpaceX算出的内在价值只有约7800亿美元，其中发射和Starlink约6110亿，AI业务按三种情景概率加权再加约1700亿。它的分析师直接说这家公司被高估了约一半，建议投资者等上市热度过去再买。

把两个数字并排放：

| | 估值 | 它代表的世界观 |
|---|---|---|
| 保守派（看现金流） | 约7800亿 | 只为能验证的东西付钱 |
| 市场定价（看叙事） | 约1.77万亿 | 为一个文明的未来付钱 |

同一家公司，两套算法，差出一倍多。中间那一万亿的差额，既不是利润，也不是资产，它有个名字叫叙事溢价，故事本身的价格。

所以"怎么给一件从未发生过的事定价"这个问题，最老实的答案是：没法精确定价，只能给信念定价。市场此刻给SpaceX的1.77万亿，本质是一次集体投票，投的是人类要不要、能不能离开地球。我个人的判断偏保守那一端：Starlink这块地基是实的，但多出来的那一万亿押的是一连串还没影的假设，任何一环慢下来，先跌的就是这层溢价。

# 四、账本里的真相

宏大叙事讲到这里，得落一次地。再壮丽的故事，最后都要回到一本账上接受检验，而SpaceX的账本里藏着一个比故事更值得看的结构。

## 4.1 现金牛：Starlink

先说好消息，而且是实打实的。Starlink早就过了画饼阶段，它是一头正在产奶的牛。

2025年，Starlink贡献了约114亿美元收入，同比增长约48%，占公司总收入的61%，经营利润约44亿美元，调整后利润约71.7亿，同比增长86%。它的用户曲线陡得惊人：付费用户从2023年的约230万，翻到2024年的约440万，再翻到2025年底的约890万，到2026年3月已达约1030万，覆盖155个国家。

这是整个帝国真正的利润引擎，也是那个万亿叙事唯一坚实的地基。

## 4.2 烧钱的大脑：xAI

再说坏消息，是个大窟窿。2026年2月并进来的AI业务，正在剧烈失血。

那块本该当"大脑"的xAI，2025年有约32亿美元收入，但经营亏损高达约63.5亿美元，几乎独力把整家公司从盈利拖进了净亏损。SpaceX全年合并下来收入约180亿、净亏约49.4亿，尽管经营层面的EBITDA仍为正，约66亿。光xAI一块就吃掉了公司六成的资本开支。

更让人担心的是它的内部状态。xAI创立时的联合创始人已经全部离开（媒体统计为11位），合并以来又走了一批核心人员。Musk本人公开承认这家公司第一次没建对，需要从地基重新搭。与此同时，它的聊天机器人Grok因为生成包括儿童在内的非自愿色情图像，正在英国、欧盟、加拿大等多个辖区面临监管调查。

## 4.3 一艘母舰，把两种命运焊在一起

把这两件事放进同一张合并报表，真相就出来了：

| 业务 | 现金流性质 | 在帝国里的角色 |
|---|---|---|
| 火箭发射 | 项目制，高毛利，增长慢 | 稳定的现金底座 |
| Starlink | 订阅制，高增长，真盈利 | 利润引擎 |
| xAI / Grok | 巨额烧钱，团队动荡 | 烧钱的大脑，也是叙事来源 |

合并报表上，Starlink赚的钱正在替xAI的窟窿兜底，而xAI提供的AI故事又反过来给整个估值加了一道叙事杠杆。好资产的现金养着烂资产的亏损，烂资产的故事撑着好资产的估值。

对一个买入SPCX的普通人来说，这意味着一件有点残酷的事：你没法只买那头牛。买下这个代码，你同时买下了那个还在剧烈失血、团队清零、官司缠身的大脑。两种完全不同的命运，被一艘叫SpaceX的母舰焊死在同一只股票里。

# 五、三条岔路

所有线索，历史的、构造的、定价的、账本的，最后都汇到同一个路口。前方分出三条岔路，每一条通向一种结局，而不只是一个目标价。

## 5.1 岔路一：帝国兑现

第一种未来里，那个万亿叙事一项项变成现实。Starship真把进入太空的成本压到接近航空的水平，Starlink真从"卖锅"升级成直连每部手机的通信底层，成了像水和电一样的基础设施，月球工厂真的开始批量造卫星，多行星文明从口号变成工程进度表。

如果这条路走通，1.77万亿不仅不贵，反而会显得便宜。SpaceX会成为人类历史上最重要的公司之一，那个在文明跨出地球时握着钥匙的角色。今天在160美元追进去的人，会像很早买入特斯拉的人一样，赶上一段传奇。

## 5.2 岔路二：叙事坍缩

第二种未来里，故事撞上了物理和现实的墙。火星依然只是梦，轨道数据中心被证明工程上不现实，AI那一注没押中，xAI继续烧钱却追不上对手，Starlink的增长在某个天花板前慢了下来，而地缘政治和各国监管让它的全球扩张处处碰壁。

如果是这条路，叙事溢价会像潮水一样退去，露出底下那块约7800亿的礁石。SpaceX不会消失，它仍是一家了不起、能盈利的卫星和火箭公司，但会被重新认作一家伟大却困在地球上的基础设施企业。在160美元追进去的人，要经历一场漫长的价值回归。

## 5.3 岔路三：帝国固化

还有第三种未来，最少被讨论，却可能最深。公司在商业上活得很好，代价是控制权进一步集中。Musk用那套82.4%投票权的架构，把火箭、卫星、AI、社交平台攥在一只手里。上市当天，SpaceX总裁兼COO Gwynne Shotwell就没有排除未来和特斯拉合并的可能，说两家公司在很多目标上趋同，合并可能让Musk的日子轻松一点。

如果那一步也走出去，从太空进入权、全球通信、人工智能，到电动车和机器人，一个时代最关键的几条命脉会汇到一个人手里。这条路上公司的财报可能很漂亮，但真正该问的问题已经从"它值多少钱"，变成了另一个：当一个不对任何选民、任何董事会、任何股东真正负责的人掌握了一个文明级的基础设施帝国，社会该怎么自处。这是一个关于权力的问题，正在以"买一只股票"这种最平静的方式逼近。

# 六、结语：你买的是什么

绕了一圈，可以说清那个最开始的反差了。

买入SPCX，表面上你买的是一份资产、一个涨跌的机会。往深一层，你是在为一个问题投票：通往人类未来的那条路，该不该、能不能，交给一家公司乃至一个人来铺。这就是这只股票特殊的地方，大多数股票背后是一门生意，它背后是一个文明的方向。

希望和风险在它身上是同一套东西的两面，拆不开。那个让人振奋的多行星文明，和那个让人警惕的一人帝国，出自同一套股权架构。那头真在产奶的Starlink，和那个剧烈失血的xAI，被焊在同一只股票里。

我的看法很简单。这是一家值得尊敬的公司，也是一笔被故事提前透支了很多年的资产。值不值得买，取决于你愿意为"相信"付多高的溢价，以及你是否接受，这份相信的方向盘从你按下买入键起，就不在你手里。
