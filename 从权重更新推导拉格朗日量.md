# 从机器学习的权重更新，一步步推出拉格朗日量

> 这份讲义只做一件事：**从你天天写的那行 `w ← w − η·∇Loss` 出发，不预设任何物理，一步步把它"翻译"成拉格朗日力学，连摩擦项、以及"有摩擦时那条诺特守恒律会怎样松开成一条衰减律"都一路推到底。**
> 主流论文通常反着写（先给一个拉格朗日量、或 Bregman Lagrangian，再说它对应某个优化器），所以读起来花里胡哨。这里是正向的、给"懂 ML 但拉格朗日忘光了"的人看的版本。
> 每一步只用一个公式，公式后面立刻翻译成大白话。哪些是学界标准、哪些是我替你做的教学重构，文末交代清楚，并附一张"本讲义每步 ↔ Tanaka-Kunin 论文方程号"的对照表。

---

## 第 0 步：起点，就一行代码

梯度下降更新权重，就是这一行：

```
w_{t+1} = w_t − η · ∇Loss(w_t)
```

翻译：**新权重 = 旧权重 − 学习率 × 梯度**。

- `∇Loss`：梯度，指向 loss **上升**最快的方向；前面加负号，就朝**下降**方向走。
- `η`：学习率，一步迈多大。

这就是全部起点。**纯 ML，没有半点物理。**

---

## 第 1 步：把"一步步跳"变成"连续地流"

上面是离散的（t、t+1，一跳一跳）。把步长想象得越来越小，看它连续的样子。先把式子两边除以 η：

```
(w_{t+1} − w_t) / η = − ∇Loss(w_t)
```

左边那个"（新 − 旧）/ 步长"，当步长 → 0，**按定义就是导数 ẇ**（w 对时间的变化率）。于是：

> **ẇ = − ∇Loss(w)**     ……（方程一，一阶）

这叫**梯度流（gradient flow）**。

⚠️ 注意：**这一步只是把更新公式取了连续极限，还是纯 ML，没有物理。而且它是一阶的（只有一个点 ẇ）。这就是你记忆里"我们那个是一阶"的来源——你没记错。**

> 文献依据：这一步是公认的。Neural Mechanics（Kunin et al., ICLR 2021）原话——梯度下降更新是这个 ODE `dθ/dt = −g(θ)` 的前向欧拉离散，η→0 时精确等于它，称为 gradient flow。

---

## 第 2 步：读懂这个一阶方程，记住它的关键性质

`ẇ = −∇Loss` 的意思：**权重的速度，直接等于下坡方向。** 坡越陡跑越快；一到平地（梯度=0）**立刻停**。

关键性质（后面对比要用）：**它没有惯性。** 像在浓糖浆里挪东西——松手立刻停，不会因为"刚才跑得快"而冲过头。这就是纯梯度下降的本来面目。

---

## 第 3 步：试着给它配一个拉格朗日量 —— 撞上第一个真实裂缝

到这里 ML 该给的都给了。物理学家看到一个运动方程，会习惯性地问：**能不能找一个标量函数 L，扔进"欧拉-拉格朗日方程"这台绞肉机，正好生出这个运动方程？**

欧拉-拉格朗日方程长这样（先不用懂，只看结构）：

```
 d/dt (∂L/∂ẇ) − ∂L/∂w = 0
```

这里藏着一个**硬事实**，也是你"一阶怎么变二阶"困惑的总根源：

> 那个 `d/dt` 作用在含 ẇ 的项上，会**再生出一个时间导数**，所以欧拉-拉格朗日方程**天生只能产出二阶方程（带 ẅ）**。

**结论：标准机械型 L = T − V 的欧拉-拉格朗日方程自然给出二阶方程，生不出一阶。** 所以你那个纯 ML 的一阶梯度下降，**不是这种标准机械型拉格朗日系统**。（严格讲，退化拉格朗日、一阶作用量、Hamiltonian、contact 几何力学这些非标准形式是能产一阶动力学的；这里走的是最主流的机械型那条路。）这不是谁搞错了，是标准机械框架和朴素梯度下降之间一条真实的裂缝。

物理学家的对策：先写一个**二阶**的、带惯性的系统（这个能配 L），再想办法让它**退化**回你那个一阶的。下一步就干这个。

---

## 第 4 步：引入二阶 —— 也就是把"惯性"和"摩擦"加进来

写一个二阶的牛顿式系统：

> **m·ẅ = − ∇Loss − μ·ẇ**     ……（方程二，二阶）

逐项翻译（每项都能在 ML 里找到对应）：

| 项 | 物理含义 | 对应 ML 里的什么 |
|---|---|---|
| `m·ẅ` | 质量 × 加速度 | **惯性**：权重多"不愿改变运动状态"。对应**动量法（momentum）**里的动量 |
| `−∇Loss` | 下坡的"力" | loss 的梯度 |
| `−μ·ẇ` | **摩擦/阻力**（永远和速度反向，越快拖得越狠） | **阻尼**：对应动量法里的 (1−β)，让更新逐步衰减 |

⚠️ **别把摩擦和 weight decay 搞混（常见错）。** 上面这个 `−μ·ẇ` 是和**速度**反向的耗散摩擦。weight decay 是另一回事：它是 `−k·w`，和**位置**成正比的保守恢复力，等价于在 loss / 势能上加一项 `(k/2)·‖w‖²`。一个是非保守的摩擦、一个是进势能的保守力，待遇完全不同，后面速查表里分开列。

**这个二阶方程，才是能写出拉格朗日量的那个。** 下面反推它的 L。

---

## 第 5 步：写出拉格朗日量 L（先不管摩擦）

先把摩擦关掉（μ=0），看保守部分：`m·ẅ = −∇Loss`。

要找的 L 是：

> **L = ½·m·‖ẇ‖²  −  Loss(w)**
> 　　 = 动能 T  −  势能 V

两个关键对应（这就是整个"反推"的命门）：

- **势能 V = Loss(w)。** 为什么？因为势能的物理意义就是"系统想往低处走的那个地形"，受的力是 −∇V。而 loss 恰恰就是你想往低处走的地形，力是 −∇Loss。**结构完全一样，所以 loss 扮演势能。**
- **动能 T = ½·m·‖ẇ‖²。** 就是"权重跑多快"，带上惯性 m。

**验证**（跟着核一遍，确认不是硬套）：
- `∂L/∂ẇ = m·ẇ`，对时间求导 → `m·ẅ`
- `∂L/∂w = −∇Loss`
- 代进欧拉-拉格朗日：`m·ẅ − (−∇Loss) = 0` → `m·ẅ = −∇Loss` ✓ 对上了。

所以 **L = ½m‖ẇ‖² − Loss(w)**。要说准确点：它不是从那行权重更新唯一推出来的，而是当你把 loss 当势能、想把一阶梯度流嵌进一个二阶保守系统时，最自然的那个机械型选择。它不空降，但也不唯一。

⚠️ 小心：**L 本身不守恒、会随时间变，这没关系——L 的职责是"生成运动方程"，不是当守恒量。** 守恒的是另一个量（见后面能量那节）。

---

## 第 6 步：把摩擦加回来 —— 当作"广义力"放在方程右边

目标是得到带摩擦的运动方程：

> **m·ẅ + μ·ẇ = −∇Loss**

关键认识：**摩擦是耗散的、不可逆的（让能量单调流失、时间倒过来不成立），它没有势能形式，所以塞不进 L = T − V。** 这不是谁的偏好，是最小作用量原理的硬限制——L 描述的是能量在动能和势能之间无损来回倒的保守世界，摩擦是往这个世界外面漏能量。

所以标准做法是：**L 保持干净的保守形式，摩擦作为一个"广义力" Q 挂在欧拉-拉格朗日方程的右边。**

> 拉格朗日量（保守，照旧）：**L = ½·m·‖ẇ‖² − Loss(w)**
> 欧拉-拉格朗日方程右边加上广义力 Q：

```
 d/dt(∂L/∂ẇ) − ∂L/∂w = Q_摩擦 ,    Q_摩擦 = − μ·ẇ
```

代进去（左边 = m·ẅ + ∇Loss）：

```
 m·ẅ + ∇Loss = − μ·ẇ
 ⟹  m·ẅ + μ·ẇ = − ∇Loss   ✓
```

**这就是连摩擦一起写完的版本：L 还是那个保守的 L，摩擦不在 L 里，而是作为非保守广义力 Q_摩擦 = −μẇ 放在方程右边。** 这正是经典力学处理非保守力的标准路子，也接得上前面的 λ：**约束力用 λ 写进 L 那一侧（约束不做功、不耗散）；摩擦这种非保守力出 L、当广义力 Q 放右边（它做负功、耗散）。约束和摩擦在拉格朗日框架里待遇不同。**

> **便捷打包（瑞利耗散函数）：** 当有很多自由度、要一次性生成所有摩擦力时，可以用一个函数 F = ½·μ·‖ẇ‖² 来统一生成：Q_摩擦 = −∂F/∂ẇ = −μẇ。F 只是"打包生成各个 Q_摩擦 的工具"，本质和上面一样——摩擦仍在 L 外面。
>
> **（旁注，不用管）** 另有一种技巧流叫 Caldirola-Kanai，用一个时间因子 e^(μ/m·t) 把摩擦硬塞进单一的 L。它的唯一用处是让"单一 L"能直接套诺特定理（TK 那篇为此用了它），代价是那个因子不对应真实能量、物理意义模糊。**我们这份讲义不用它**——我们用上面"保守 L + 广义力 Q"的写法，更透明，而且下一节会看到，有广义力时诺特定理照样能推。

---

## 第 7 步：闭环 —— 让二阶塌回你最初的一阶梯度下降

现在解决你最大的疙瘩：ML 明明是一阶，怎么冒出二阶？答案在这步闭合。

让惯性小到可以忽略（质量趋零 m→0，也就是阻尼相对惯性极大的"过阻尼"极限）。在 `m·ẅ = −∇Loss − μ·ẇ` 里，`m·ẅ` 这项相对另外两项小到可以扔掉，剩下：

```
0 ≈ −∇Loss − μ·ẇ   ⟹   ẇ = −(1/μ)·∇Loss
```

**这正是第 1 步那个一阶梯度下降的形式 ẇ = −(1/μ)·∇Loss。** 它和第 1 步的梯度流只差一个时间尺度（第 1 步已经把 η 吸进时间里、不再单独出现了）。如果把离散步的编号当时间看，这个 1/μ 可以类比"有效学习率"。两处的口径不一样，别直接把它和第 1 步的 η 划等号。

整条链闭合了：

```
ML 的一阶梯度下降（第1步，ẇ = −∇Loss）
        │  物理为了能写 L，一般化成……
        ▼
二阶带惯性带摩擦的牛顿系统（第4步，m·ẅ = −∇Loss − μ·ẇ）
        │  保守 L = ½m‖ẇ‖² − Loss，摩擦作为广义力 Q=−μẇ 放方程右边（第5、6步）
        │  把摩擦 μ 开到极大、惯性 m 忽略……
        ▼
又塌回一阶梯度下降（第7步，ẇ = −(1/μ)∇Loss，1/μ 类比有效学习率）
```

**一句话：一阶（梯度下降）和二阶（带惯性的小球）不矛盾。二阶是更大的家；一阶是它"惯性趋零、过阻尼"那个角落。而 ML 里真实的"动量法（momentum SGD）"，恰恰是把惯性 m 加回来、不取那个极限——它就活在二阶的世界里。所以二阶不是虚构的，它对应一个你天天用的优化器。**

## 配图：一眼看懂"守恒 = 学不会，耗散 = 才能学"

（见同时给你的图 `learning_as_dissipation.png`）

我拿一个最简单的 loss：`Loss(w) = ½(w−3)²`（一个谷底在 w=3 的山谷），跑了三种情况：

**左图（权重轨迹 w 随时间）：**
- 🔴 **无摩擦（μ=0）**：权重在谷底两边**永远来回震荡**，0 ↔ 6 荡个不停，**永远停不到 w=3**。→ 学不会。
- 🔵 **有摩擦（欠阻尼）**：阻尼振荡，慢慢**收敛到 w=3**。→ 学会了，还带点惯性。
- 🟢 **重摩擦（≈梯度下降）**：单调地爬向 w=3，不震荡。→ 这就是纯梯度下降。

**右图（总能量 E = T + V 随时间）：**
- 🔴 无摩擦：能量是**一条平线**（守恒，dE/dt = 0）。→ 它把"loss 降下去的势能"全换成了动能，所以冲过头、停不下。
- 🔵🟢 有摩擦：能量**单调往下掉**（dE/dt < 0，耗散）。→ 多余的动能被摩擦吃掉，小球才能停在谷底。

**这张图就是整串结论的画面版：**

> 在这个二次 loss 的例子里：落到谷底 = 把能量耗散掉。能量守恒（无摩擦）的系统会一直在谷底两边震荡、**停不下来**；只有能量被耗散（有摩擦），系统才停得到谷底。所以在这套连续优化的图像里，**收敛到极小点要靠耗散，纯保守系统通常只会绕着谷底打转**，这不是比喻，是能算出来、画得出来的。（一维二次 loss 看得最干净，一般非凸情形会更复杂。）

## 第 8 步：诺特定理，先讲保守版，再讲它的广义版

这是整串最关键的一步。前面已经把摩擦请进了运动方程，现在要问：摩擦会怎么改写"对称对应守恒量"这条物理学里最漂亮的定律。这一节分三遍讲，第一遍只讲教科书里那个干净的诺特定理，先把它讲透；第二遍把摩擦加进去，看守恒怎么变成衰减；第三遍再落回学习。

### 第一遍：经典诺特定理（保守世界里那条漂亮定律）

物理系学生背得最熟的一句话：**每一个让作用量保持不变的连续对称性，都对应一个守恒量。** 空间平移不变对应动量守恒，时间平移不变对应能量守恒，旋转不变对应角动量守恒。简洁又漂亮。

放进我们这套"权重当小球"的语言里，对称就是 loss 沿某个方向 δw 是平的：你把权重往这个方向挪，loss 一点不变。诺特定理说，这种平方向上有一个量会被守住。这个量叫诺特荷，定义是沿 δw 方向的那份"动量"：

> **Q = ⟨ ∂L/∂ẇ , δw ⟩ = ⟨ m·ẇ , δw ⟩**

为什么它守恒？算它的时间导数，用乘积法则拆成两项：

```
 dQ/dt = ⟨ d/dt(∂L/∂ẇ) , δw ⟩ + ⟨ ∂L/∂ẇ , d(δw)/dt ⟩
```

在保守世界里，欧拉-拉格朗日方程是干净的 `d/dt(∂L/∂ẇ) = ∂L/∂w`。代进去，再用"loss 沿 δw 平"（保守部分的项加起来正好为零），整个右边消成零：

> **dQ/dt = 0 ，Q 守恒。**

这就是教科书版诺特定理。漂亮，干净，闭合。但它脚底下踩着一个前提，等会儿要紧：**这条推导从头到尾用的是没有摩擦的欧拉-拉格朗日方程，系统必须是保守的，所有的力都能从 L 里推出来。** 一旦有了摩擦、辐射阻尼这类耗散力，干净的推导就断在第一步。

### 第二遍：广义诺特定理（把耗散加回来，守恒变衰减）

现在把摩擦加回来。欧拉-拉格朗日方程右边多了广义力（第 6 步那个 Q_摩擦）：

```
 d/dt(∂L/∂ẇ) = ∂L/∂w + Q_摩擦 ,    Q_摩擦 = −μ·ẇ
```

把这个带摩擦的方程，代进刚才 dQ/dt 的第一项。保守部分还是像第一遍那样靠对称性消掉，但这次多留下了摩擦那一项，消不掉了：

```
 dQ/dt = ⟨ Q_摩擦 , δw ⟩ = ⟨ −μ·ẇ , δw ⟩ = −μ·⟨ ẇ , δw ⟩
```

在欧几里得情形下，⟨ẇ, δw⟩ 正比于 Q 本身（因为 Q = m⟨ẇ, δw⟩，所以 ⟨ẇ,δw⟩ = Q/m）。代进去，右边就化成 Q 的一个倍数。我们把这个倍数记成 λ，它就是这套系统的**耗散强度**：

> **dQ/dt = −λ·Q ，其中 λ = μ/m。**

解这个一阶方程，Q 会一路指数衰减下去：

> **Q(t) = Q(0)·e^(−λt)**

现在把两遍并排放：

> **λ = 0（无摩擦、保守）：dQ/dt = 0，诺特荷守恒。**
> **λ ≠ 0（有摩擦、耗散）：dQ/dt = −λ·Q，诺特荷按 e^(−λt) 衰减到零。**

并排一放，就看出一件比单看任何一遍都更深的事。教科书那条"对称对应守恒量"，只是这个广义版本在 λ=0 时缩成的一个特例。**真正普遍成立的是那条衰减律，守恒不过是把耗散强度调到零、回到理想无摩擦世界时，衰减率恰好变成零的极限。** 我们一直当成天经地义的守恒定律，其实是一条衰减定律在 λ→0 时露出的尾巴。对称性自始至终都在（loss 沿那个方向一直是平的），是 λ 决定了这份对称到底兑现成"守住"还是"漏光"。

> ⚠️ **限定一下，免得被挑刺。** 这条干净的指数衰减 dQ/dt=−λQ，只在**欧氏动能 + 各向同性线性阻尼 + 简单（参数无关）的对称方向**下成立。它隐含一个前提：对称生成元 δw 是常向量（平移型），所以 ⟨ẇ,δw⟩=Q/m 这步才对。而像 ReLU 重缩放那种对称，δw 依赖参数本身（δw ∝ (w_in, −w_out)），事情要复杂得多，这正是 Tanaka-Kunin 真正研究的**动能对称破缺（kinetic symmetry breaking）**，不是一句"摩擦让守恒量按 e^(−λt) 衰减"能盖住的。这里的 λ 衰减请当成欧氏教学版，不是一般神经网络对称性的完整结论。

### 第三遍：落到学习上，理想学习系统就像惯性参考系

最后把这套搬进深度学习，刚才那个 λ 就有了名字。

先想一个"理想学习系统"，它在学习的世界里相当于物理里的惯性参考系：要么 loss 是个常数（没有坡度，权重不被往任何方向拽），要么耗散强度 λ=0（没有摩擦）。这种系统不耗散，能量守恒，诺特荷一找一个准，老老实实守着。代价是它也永远学不会，只会在谷底两边震荡，前面那张图里的红线就是它。

真正的学习一开就变了。**把 λ 从零拧大，等于在权重空间里凭空多出一个摩擦样的非保守场**，系统从保守变成耗散，前面那份漂亮的对称和守恒一起被破坏。可这不是坏消息。我们照样能写出理想系统里那个诺特荷 Q，只是它现在不再守恒，开始按 e^(−λt) 衰减，**而这个衰减本身就是学习过程**。摩擦不光推着权重沿有坡的方向下山，还顺手把那些对称平方向上残留的乱晃一点点收干净。

所以真实训练（带 weight decay、带阻尼、有限学习率）里，那些理论上该守恒的量（比如 ReLU 的 ‖W_in‖²−‖W_out‖²）会被慢慢拉向某个值，衰减率就是那个 λ，由你的阻尼、动量、正则一起定。守恒荷的衰减，就是"学习连没用的对称方向上的多余漂移也一起收拢"的精确写照。

## 附二：两篇相关论文导读，以及三种 decay 的统一图像

### 先说定位：读这两篇是为了检验，不是为了重学

到这里，我们手上已经有一个完整的框架了。理想的守恒情形，是 loss 为常数（地形全平、没在学）的那种系统。真实的学习，是"一个 loss 势能，再加上一个非保守的力场"。在这个框架里，诺特量在学习过程中一定会衰减，而守恒只会发生在 loss 平、也就是没在学的那些死方向上，所以对学习本身没有意义。

下面这两篇论文，是这条线上最该读的两篇。读它们的目的很单纯，检验一下它们会不会动摇上面这个图像。结论先摆出来：都不动摇，我们的图像原样不变。下面逐篇说为什么。

### 第一篇：Tanaka & Kunin, Noether's Learning Dynamics（NeurIPS 2021）

先讲它在做什么。这篇的核心是一个统一方程，论文里叫 NLD（Eq.11），它描述的事情和我们一样：神经网络里的诺特荷不守恒，而是会"运动"。

它的前半段，和我们这份讲义几乎逐句重合。把学习写成拉格朗日、指出梯度下降是一阶而牛顿力学是二阶（那道"裂缝"）、再用有限学习率带来的隐式加速把一阶补成二阶，这些就是我们第 1 到 7 步做的事，对我们没有新东西。

它真正原创的一点，叫动能对称破缺（kinetic symmetry breaking，简称 KSB）。我们之前只盯着 loss（势能）对不对称，TK 提醒：还得看动能（也就是学习规则那一项）对不对称。举个最清楚的例子，缩放对称（BatchNorm 就带这种对称）下，把权重缩放一下，动能 ½|q̇|² 会变成 (1+s)²|q̇|²，它依赖缩放量 s，也就是说动能本身破坏了这个对称。这就是 KSB，正是它让诺特荷动了起来。

它的杀手级应用很漂亮：证明了"带 BatchNorm 的网络用普通 SGD 训练"，约等于"一个用 RMSProp 这种自适应优化器训练的网络"，一句话就是 BatchNorm ≈ RMSProp。

我们怎么看它。说白了这是一次 cherry-picking，它挑了一个特定的对称（缩放，对应 BatchNorm）加一个特定的算法配置，在这个特例里证出一个等价。换个网络、换个对称、换个优化器，结论就变。所以它不是关于"整个学习过程"的普适定律，而是一块拼图式的局部结论。它没有推翻我们的三种 decay 图像，只是把其中某一项，算到了一个具体的特例里。

### 第二篇：Ziyin et al., Parameter Symmetry and Noise Equilibrium of SGD（NeurIPS 2024）

先补一句最基础的，SGD 是什么、为什么说它"带噪声"。普通梯度下降 GD 用全部数据算出精确的平均梯度，是确定性的。SGD 每一步只随机抽一小批（batch）数据来算梯度，所以每步的梯度等于真实梯度，加上一个因为抽样带来的随机误差，也就是噪声。用物理语言说，这正好是一个朗之万方程：一个确定的梯度力，加上一个随机的扰动力（布朗运动那种）。

它的方法是纯数学，随机微积分。把 SGD 写成朗之万随机微分方程（论文 Eq.1），再用 Itô 引理去求诺特量的变化。关键在于，Itô 引理会比普通求导多出一个二阶项，最后得到 Ċ = σ²·Tr[Σ·∇²C]（Eq.5）。纯 GD 没有随机项，也就没有这个二阶项，所以诺特量守恒；一旦加上噪声，二阶项冒出来，它就不守恒了。

它的核心结论叫噪声平衡点（noise equilibrium）。纯 GD 下，诺特荷在对称方向上守恒，卡在初始值不动。一加噪声，它就被推着走，最后收敛到一个唯一的、有吸引力的、与初始化无关的固定点。作者用这个点解释了一串现象：progressive sharpening 和 flattening、warmup 为什么有用、深层网络各层范数自动趋于平衡、以及 SGD 和 GD 为什么学到不同的表征。

提一句作者背景。这是物理出身的人（Ziyin、Ueda 这一脉）拿统计物理里的随机过程工具，来研究 SGD 这个算法。要注意分清：他用的是物理的方法，但研究对象是算法、不是物理；目标是"刻画这个算法的行为"，不是"定义学习的本质"。

我们怎么看它。他从头到尾讨论的，都是那个"loss 平、没在学"的死方向。纯 GD 下它守恒，加噪声后它漂到一个固定点，可自始至终，这都是那个不影响 loss、和学习无关的方向。所以无论他发现的是守恒还是噪声平衡，对我们"学习就是诺特量衰减"这件事都没有意义。他讨论的方向，和我们关心的方向（在学的、有坡度的方向），是平行的、互不相干的两条线。

### 一个表面矛盾的解开

这里有个特别容易绕晕的地方，得专门说清楚。

表面上看，"Ziyin 说 GD 下诺特荷守恒"和"我们说学习中诺特量衰减"，好像直接打架。其实一点都不矛盾。诺特量到底守不守恒，只取决于一件事：有没有一个力去推那个对称方向。

- 纯 GD（只有 loss 梯度）：loss 的梯度恰好不碰对称方向，没有力去推它，所以守恒（这就是 Ziyin 说的）。但守恒的这个方向，正是"没在学"的死方向，所以对我们没意义。
- GD 加非保守力（摩擦、weight decay）：有力在推它，于是衰减（这是我们的框架）。
- SGD 加噪声：噪声随机地推它，于是漂到一个固定点（这是 Ziyin 这篇）。

三种情况，三个结果：纯 GD 守恒、加摩擦衰减、加噪声漂移。它们不矛盾，差别只在"往那个方向加了什么力"。而且 Ziyin 那个"守恒"恰恰替我们做了印证：守恒只发生在没在学的死方向上。

### 三种 decay 的统一图像

这一节是重点，我们从第一性原理把它摆开。诺特荷的定义是 Q = ⟨∂L/∂ẇ, δw⟩，也就是广义动量在对称方向 δw 上的投影。不做任何简化，直接求它对时间的导数，会发现它的变化有三个互相独立的来源，对应三种 decay。

- **decay 1：非保守力做功（A2 项）。** 当广义力是摩擦或阻尼 Q_力 = −μẇ 时，它给 Q 的变化贡献一项 −μ⟨ẇ, δw⟩。这就是我们第 8 步那个干净的指数衰减 dQ/dt = −(μ/m)Q 的来源。
- **decay 2：动能对称破缺 KSB（来自 A1 里的动能部分）。** 平移、旋转这类对称下它等于零；缩放对称（BatchNorm）下它不为零。这正是 TK 那篇的核心。
- **decay 3：非欧度量项（C 项）。** 用欧氏度量（普通梯度下降）时它等于零；用自然梯度、mirror descent 这类非欧优化器时它不为零。

把这三项摆出来，几件事就清楚了。

我们讲义第 8 步推出来的那条纯指数衰减，是在"平移对称（让 decay 2、以及'对称方向随时间变'那一项都归零）加欧氏度量（让 decay 3 归零）"这个最干净的特例下，三项塌成一项的结果。我们没推错，只是选了最简单的情形开刀。

TK 做的，其实就是 decay 2，他挑的是缩放对称。Ziyin 做的则是另一种东西：噪声驱动，对应前面那个 Itô 二阶项 σ²·Tr[Σ·∇²C]。它不在上面这三项里，是"随机性"带来的第四种来源，但它作用在死方向上，对我们没意义。

最后用一个物理画面收一下。想象一个环形的山谷，谷底连成一个圈，沿着这个圈绕（环向）就是那个对称的死方向，loss 在这个方向上是平的。纯 GD 时，球在环向不受力，不动，这是守恒。加上摩擦，球被沿着环向拖动、能量被耗散，这是 decay 1。加上噪声，球被随机地乱推，最后停在一个平衡位置，这是 Ziyin 的噪声平衡。同一个环向方向，加不同的力，行为就不同。

### 本节结论

两篇论文，既没有动摇、也没有补充我们的核心图像。框架原样不变：理想守恒等于 loss 为常数；真实学习等于 loss 势能加一个非保守力场；诺特量在学习中必然衰减；守恒只发生在没在学的死方向上，没有意义。TK 是 decay 2 的一个特例（cherry-picking 出来的），Ziyin 研究的是死方向上的噪声平衡（和学习无关）。我们真正关心的，是"在学的、有坡度的方向"上诺特量怎么衰减，这正是后面要继续往下推的、三种 decay 的统一刻画。

### 哪些是他们的，哪些是我们的整理

延续这份讲义一贯的诚实口径，把账分清楚。TK 的 KSB、BatchNorm ≈ RMSProp，以及 Ziyin 的朗之万写法、Itô 二阶项、噪声平衡点，这些结论都是他们的。把它们归进"三种 decay 的统一图像"这张表里、并判断"两篇都不影响我们的框架"，是我们这边的整理和判断，不是论文里现成的话。
