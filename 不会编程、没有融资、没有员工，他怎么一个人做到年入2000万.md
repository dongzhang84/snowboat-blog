

先讲一个故事。

一个荷兰小伙子，大学读的是音乐制作，毕业后在阿姆斯特丹做电子音乐DJ，顺便做YouTube频道。频道做得还不错，但赚不到什么钱。

音乐这条路走不通，他想转行。但他没有任何技术背景，不会写代码，专业背景和计算机完全不沾边。

快30岁的时候，他决定自学编程。没上过一天编程课，全靠网上的免费教程，从最基础的PHP开始学起。

学了一段时间之后，他做了一件在当时看来很疯狂的事情：**公开宣布要在12个月内做12个产品**。一个月一个，不是原型不是概念，而是有支付系统的、可以真正上线赚钱的完整网站。

他把这个挑战发到了网上，公开记录每一天的进度。

然后呢？

**然后绝大多数都失败了。**

12个产品做完，几乎全部没有人用，没有人付钱。他没有放弃，继续做，后来几年里一共做了**70多个项目**。70多个，想想这个数字。大多数创业者一辈子可能就做一两个项目，他做了70多个，其中66个是失败的。

但有**4个**活了下来。

就这4个，让他现在一个人年入超过**300万美元**（约2000万人民币），利润率超过90%。零员工，零融资，零办公室，全世界到处跑，在咖啡馆里写代码。

他叫**Pieter Levels**，网名levelsio，1987年出生，是全世界独立开发者圈子里公认的标杆人物。上一篇我写了MEDVi那个4亿美元营收的骗局。Levels是MEDVi的反面：每一分钱都干干净净，经得起审计。

**我花了很长时间拆解他的方法论，发现他做对的事情其实并不复杂。以下是我总结的五条核心方法，每一条普通人都可以照着做。**


## 产品矩阵：七个产品，一个人跑

Levels现在同时运营着七个产品。按收入从高到低：

**PhotoAI（AI生成照片）**
- 2023年2月上线，用AI帮用户生成各种场景的个人照片（职业照、社交照、约会照等）
- 上线第3周就到了1万美元MRR
- 18个月后到了13万美元MRR
- 目前年化收入约**160万美元**，是他最赚钱的单品
- 赶上了2023年AI图片生成的风口，Levels自己说"时机比什么都重要"

**NomadList（数字游民城市数据库）**
- 2014年做的，他的第一个成功产品
- 帮远程工作者选城市：WiFi速度、生活成本、天气、安全指数等几千个数据点
- 付费会员制，从最初的5美元一次性付费，逐步涨到75美元/年
- 目前年收入约**50万到60万美元**

**RemoteOK（远程工作招聘板）**
- 2015年上线，面向企业的远程职位发布平台，B2B模式
- 企业发一条招聘帖收费几百美元
- 年收入约**40万到50万美元**
- 比NomadList后来居上，因为B2B的客单价比B2C高得多

**其他产品**：Interior AI（AI室内设计）、Avatar AI（AI头像）等，收入较小但依然盈利。

所有产品加起来，年收入超过**300万美元**，利润率超过**90%**。


## 他做对了什么：五条可复制的方法论

### 第一条：用最无聊的技术栈

Levels的技术栈是：PHP + jQuery + SQLite。没有React，没有Next.js，没有任何现代前端框架。

这在程序员社区里经常被嘲笑。但Levels的回答很简单：他2014年学的PHP，用了十年，所有七个产品都用同一套代码模式。一个人要同时维护七个产品，不可能每个都用不同的技术栈。

他所有的网站都跑在一台**40美元/月的VPS服务器**上。

核心逻辑：**技术栈的好坏不取决于它有多新，而取决于你用它的速度有多快**。Levels用PHP比任何人用React都快，因为他用了十年。

### 第二条：12个月做12个产品

大多数创业者的问题是：花太长时间在一个想法上，最后发现市场不买账。

Levels的方法是反过来：**快速试错，让市场告诉你哪个对**。一个月做一个，做完就上线，有人付钱就继续做，没人付钱就下一个。

他做了70多个项目，只有4个赚到了钱。但这4个就足够了。

### 第三条：Build in Public

Levels从2014年开始就在Twitter上公开分享他的收入、代码、产品数据、失败经历。他是最早一批"公开建设"的独立开发者。

这样做的好处是：

1. **免费获客**。他的每一条推文都是一次免费的产品曝光。他不花一分钱做广告，所有流量来自社交媒体和SEO
2. **建立信任**。当一个人持续十年公开展示自己的收入和失败，用户会觉得这个人是真实的
3. **吸引同频的人**。他的X账号有几十万粉丝，全是独立开发者、远程工作者、数字游民。这些人既是他的用户，也是他产品的传播者

### 第四条：程序化SEO

Danny Postma做HeadshotPro用了这个方法，其实Levels更早就在用了。

以NomadList为例。Levels用代码自动生成了几千个城市页面，每个页面都有独立的数据（WiFi速度、生活成本、安全指数等）。这些页面自然地被Google索引，带来了大量的长尾搜索流量。

不需要写文章，不需要做内容营销，只需要让数据自动生成页面。**一次写代码，永久获取流量**。

### 第五条：B2C先验证需求，B2B再放大收入

NomadList是B2C（面向个人），收费低但验证了"远程工作者需要选城市"的需求。

RemoteOK是B2B（面向企业），基于同一个市场但收费高得多。

PhotoAI也是B2C起步，但正在向B2B扩展（企业团体照、品牌素材等）。

这个顺序很重要：**先用低价B2C快速验证市场有没有需求，确认有之后再做高价B2B放大收入**。


## 他和MEDVi的根本区别

| | Pieter Levels | MEDVi（Matthew Gallagher） |
|---|---|---|
| 启动资金 | 接近零 | 2万美元 |
| 员工 | 0 | 2人 |
| 年收入 | 约300万美元 | 约4亿美元 |
| 利润率 | 90%以上 | 16.2% |
| 获客方式 | SEO + 社交媒体，零广告费 | Facebook广告 + 800个假医生账号 |
| 产品本质 | 自己写的软件产品 | 复方药品分销渠道 |
| 法律风险 | 零 | FDA警告信 + 多起诉讼 + 个人担保 |
| 可持续性 | 做了10年，还在增长 | 做了2年，正在崩塌 |

核心区别不在于规模（Levels收入远小于MEDVi），而在于**结构的健康度**。

Levels的300万美元，90%以上是利润，产品是自己写的代码，获客是免费的SEO和社交媒体，没有任何法律风险，没有任何外部依赖。即便明天所有AI公司倒闭，NomadList和RemoteOK照样运转。

MEDVi的4亿美元，16.2%的利润率，依赖Facebook广告（随时可能被封号）、依赖OpenLoop的医生网络（已经出了数据泄露）、依赖FDA的监管灰色地带（已经收到警告信）。任何一个环节断裂，整个业务归零。

**一人公司的天花板不是收入规模，而是结构的抗风险能力。**


## 普通人能学到什么

Levels的故事对普通人最有价值的不是"他赚了多少钱"，而是以下几点：

**第一，不需要CS学位。** Levels是音乐专业出身，快30岁才学编程。他用的PHP + jQuery是公认最简单的技术栈。在2026年，有了AI编程助手（Cursor、Claude Code），技术门槛比他当年低了10倍。

**第二，不需要一个完美的想法。** 他做了70多个项目，只有4个赚到钱。关键不是想法有多好，而是试的速度有多快。一个月做一个，三个月没人付钱就放弃，这比花一年做一个"完美产品"然后发现没人要好得多。

**第三，不需要融资。** Levels从来没融过一分钱。他的成本是一台40美元/月的服务器加上自己的时间。在2026年，加上几个AI工具的订阅费，一个月总成本可能不到100美元。

**第四，不需要辞职。** NomadList最早就是一个side project。Levels在做音乐和YouTube的同时开始写代码。直到NomadList稳定盈利之后他才全职做这件事。

**第五，需要时间。** 这是最不性感但最重要的一点。Levels从2014年做NomadList到现在已经超过10年。PhotoAI的爆发也建立在他10年积累的粉丝和声誉之上。没有人会在第一天就年入300万美元。但如果你从今天开始，用AI工具把"12个月做12个产品"的周期压缩到"3个月做12个产品"，你的时间表可能比Levels当年快得多。


## 结论

MEDVi告诉你一人公司能做到多大，也告诉你红线踩多了会怎样。

Levels告诉你一人公司能做到多久，也告诉你干净的钱是怎么赚的。

如果你想做一人公司，不要学MEDVi的规模，要学Levels的结构：**自己写产品、免费获客、零外部依赖、90%利润率、十年可持续**。

300万美元不够刺激？但它是你的。干干净净，没有一个假医生，没有一封FDA警告信，没有一条诉讼。

这才是一人公司该有的样子。
